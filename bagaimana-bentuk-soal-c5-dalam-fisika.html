3. ubahlah bilangan heksadesimal C5 ke dalam bentuk desimal​


Jawaban singkat:

C5 (heksadesimal) = 197 (desimal)

Penjelasan panjang:

Berdasarkan tabel sistem bilangan, diketahui bahwa

C (heksadesimal) = 12 (desimal)

Perhitungan konversi heksadesimal ke desimal (sesuai soal di atas):

C5 = 12x16¹ + 5x16⁰ = 192 + 5 = 197

Jadi C5 (heksadesimal) = 197 (desimal)

Bentuk sederhana dari [tex]\rm 5^2 \times 5^4[/tex] adalah [tex]\rm 5^6[/tex].

[ Pembahasan ]

Eksponen atau perpangkatan adalah bentuk perkalian berulang dari suatu bilangan. Yang mana banyak pengulangannya sebanyak pangkatnya.

 - Sifat Sifat Perpangkatan -

[tex]\rm a^m \times a^n = a^{m + n}[/tex][tex]\rm a^m \div a^n = a^{m - n}[/tex][tex]\rm a^0 = 1[/tex][tex]\rm \left (a^m \right )^n = a^{m\times n}[/tex]

~~~~~

Diketahui :

⟩ 5² × 5⁴

Ditanya :

⟩ Bentuk sederhana ?

Jawaban :

Gunakan sifat pertama. Maka :

[tex]\rm 5^2\times 5^4\\\rm 5^{2 + 4}\\\rm 5^6[/tex]

Maka, bentuk sederhananya adalah [tex]\rm 5^6[/tex]

33. Bentuk sederhana dari a3.b-2.c5:a-2.b7.c4 adalah


Kategori Soal : Matematika - Bentuk Pangkat
Kelas : X (1 SMA)
Pembahasan :
pᵃ x pᵇ = pᵃ ⁺ ᵇ
pᵃ : pᵇ = pᵃ ⁻ ᵇ
(pᵃ)ⁿ = pᵃ ˣ ⁿ

Mari kita lihat soal tersebut.
(a³ x b⁻² x c⁵) : (a⁻² x b⁷ x c⁴)
= (a³ : a⁻²) x (b⁻² : b⁷) x (c⁵ : c⁴)
= a³ ⁻ ⁽⁻²⁾ x b⁻² ⁻ ⁷ x c⁵ ⁻ ⁴
= a⁵ x b⁻⁹ x c¹
= a⁵ x c¹ : b⁹

Semangat!
