20. I ... complete the task. (Future Tense)a. Have completed b. Have complete c. Will completed d. Will complete#BantuJawabDong...​


Simple Future Tense

I ... complete the task.

=> d. Will complete  

PEMBAHASAN:

Simple Future tense adalah tenses yang digunakan untuk menyatakan bahwa suatu aksi terjadi dimasa depan, secara spontan atau terencana.  

Keterangan Waktu:  

Tomorrow… => Besok…  Tomorrow morning => Besok pagi  Next… => … yang akan datang  Next time => Waktu yang akan datang  Next week => Minggu depan  Tonight => Malam ini  Soon => Segera  Later => Nanti  Two days later => Dua hari lagi  Two more days => Dua hari lagi  By and by => Sebentar lagi  The day after tomorrow => Besok lusa  

Untuk menyatakan kegiatan atau rencana yang akan kita lakukan di masa depan kita dapat menggunakan:

Will  

Fungsi:  

Untuk mengajukan permohonan/permintaan.  Untuk menunjukkan keinginan atau ketidakinginan.  Untuk membuat janji.  Untuk menunjukkan fakta umum  

Formula:  

(+) Subject + shall/will + Verb I (infinitive)  (-) Subject + shall/will + not + Verb I (infinitive)  (?) Shall/will + Subject + Verb I (infinitive)  

Be Going To  

Fungsi:  

Untuk menunjukkan niat atau tujuan.  Untuk membicarakan kejadian yang akan segera terjadi.  Untuk membicarakan rencana yang telah dibuat.  

Formula:  

(+) Subject + To Be (is, am, are) + Going To + Verb I (infinitive)  (-) Subject + To Be (is, am, are) +not + Going To + Verb I (infinitive)  (?) To Be (is, am, are) + Subject + Going To + Verb I (infinitive)?

36. Benny, Denny, Jenny and Penny competed against each other during a Math challenge. The average number of questions completed by the 4 of them was 81. The number of questions completed by Benny was 2/3 the number of questions completed by Denny. Jenny completed 6 more questions than Denny. Penny completed 78 questions. How many questions did Jenny complete?


Let's use algebra to solve the problem. Let's assume the number of questions completed by Denny is "D."

We are given the following information:

1. The average number of questions completed by all four of them is 81. So, the sum of their completed questions is 81 * 4 = 324.

2. Benny completed 2/3 the number of questions completed by Denny. So, Benny completed (2/3)D questions.

3. Jenny completed 6 more questions than Denny. So, Jenny completed D + 6 questions.

4. Penny completed 78 questions.

Now, let's add up the number of questions completed by each person:

Benny + Denny + Jenny + Penny = (2/3)D + D + (D + 6) + 78

According to the first piece of information, the total number of questions is 324:

(2/3)D + D + (D + 6) + 78 = 324

To solve for D, let's combine like terms:

(2/3)D + D + D + 6 + 78 = 324

(8/3)D + 84 = 324

Now, let's isolate D:

(8/3)D = 324 - 84

(8/3)D = 240

Next, divide both sides by 8/3 to solve for D:

D = 240 / (8/3)

D = 240 * (3/8)

D = 90

Now that we have the number of questions completed by Denny (D = 90), we can find the number of questions completed by Jenny:

Jenny = D + 6 = 90 + 6 = 96

So, Jenny completed 96 questions.
