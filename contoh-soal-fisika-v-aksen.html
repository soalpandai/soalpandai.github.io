5. Selasaikan persamaan diferensial dibawah ini XY aksen + Y = 0 : Y(1) = 1 Soal selanjutnya Y aksen = X/Y : Y (2) = 0


Untuk kedua soal, saya akan ubah [tex]y'[/tex] menjadi [tex]\frac{dy}{dx}[/tex] demi kemudahan, karena saya sudah biasa seperti itu.

1.)
[tex]\displaystyle x\frac{dy}{dx} + y = 0 : y(1) = 1[/tex]

[tex]\displaystyle x\frac{dy}{dx} = -y[/tex]

Pindahkan seluruh variabel y ke kiri, dan variabel x ke kanan
[tex]\displaystyle \frac{1}{y} \ \frac{dy}{dx} = -\frac{1}{x}[/tex]

Ambil integral dx pada sisi kiri dan kanan (selalu masukan konstan pada sisi x):
[tex]\displaystyle \int \frac{1}{y} \ \frac{dy}{dx} \ dx = -\int \frac{1}{x} \ dx[/tex]

[tex]\displaystyle \int \frac{1}{y} \ dy = \int -\frac{1}{x} \ dx[/tex]

[tex]\displaystyle \ln{y} = -\ln{x} + C[/tex]

Informasi di soal menyatakan bahwa y(1) = 1, maka saat x = 1, y = 1. Gunakan informasi ini untuk mencari nilai C seperti berikut:
[tex]\displaystyle \ln{1} = -\ln{1} + C[/tex]

maka:
[tex]\displaystyle C = 0[/tex]

Lalu, kembali lagi kepada persamaan awal, namun dengan jumlah C = 0
[tex]\displaystyle \ln{y} = -\ln{x}[/tex]

cari nilai y:
[tex]\displaystyle y = e^{-\ln{x}}[/tex]
[tex]\displaystyle y = x^{-1} = \boxed{\frac{1}{x}}[/tex]


2.)
[tex]\displaystyle \frac{dy}{dx} = \frac{x}{y} : y(2) = 0[/tex]

Pisahkan variabel y ke sebelah kiri, dan variabel x ke sebelah kanan:
[tex]\displaystyle y \frac{dy}{dx} = x[/tex]

Ambil integral dx pada kedua sisi:
[tex]\displaystyle \int y \ \frac{dy}{dx} \ dx = \int x \ dx[/tex]

[tex]\displaystyle \int y \ dy = \int x \ dx[/tex]

[tex]\displaystyle \frac{1}{2}y^2 = \frac{1}{2}x^2 + C[/tex]

Gunakan informasi soal bahwa y(2) = 0 untuk mencari nilai dari C. Dalam soal ini:
[tex]\displaystyle \frac{1}{2}(0)^2 = \frac{1}{2}(2)^2 + C[/tex]

[tex]\displaystyle 0 = 2 + C[/tex]:
[tex]C = -2[/tex]

Lalu, masukan nilai C = -2 kedalam persamaan tadi:
[tex]\displaystyle \frac{1}{2}y^2 = \frac{1}{2}x^2 - 2[/tex]

Cari nilai dari y:
[tex]\displaystyle y^2 = x^2-4[/tex]

[tex]\displaystyle y = \boxed{\pm \sqrt{x^2 - 4}}[/tex]

f'(3)f'(2)f'(1)f'([tex]-\frac{1}{3}[/tex])

Jawab :

Teorema Turunan

Apabila diketahui [tex] f(x) = ax^n[/tex], maka turunannya (f'(x)) adalah

[tex]\boxed{f'(x) = an \times x^{n - 1}}[/tex]

• Cari nilai f'(x)

[tex]f(x) = \frac{3}{ {x}^{2} } \\ f(x) = 3 \times {x}^{ - 2} \\ f'(x) = 3 \times ( - 2) {x}^{ - 3} \\ f'(x) = - 6 \times {x}^{ - 3} \\ f'(x) = - \frac{6}{ {x}^{3} } [/tex]

Cara mencari nilai f'(3), f'(2), f'(1), f'([tex]-\frac{1}{3}[/tex]) tertera di lampiran (gambar)

Kesimpulan :Jadi nilai f'(3) adalah [tex] -\frac{2}{9}[/tex]

Jadi nilai f'(2) adalah [tex] -\frac{3}{4}[/tex]Jadi nilai f'(1) adalah -6Jadi nilai f'([tex]-\frac{1}{3}[/tex]) adalah 162
