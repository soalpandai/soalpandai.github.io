29. M100 N 531 Sebuah bingkisan seberat 100 N digantung di dengan dua tali da berada dalam kesetimbangan seperti pada gambar. Tentukanlah nilai tegangan tali 1, tegangan tali 2 dan tegangan tali 3! ​


Jawaban:

Tegangan tali 1 dan 2 masing-masing menahan gaya gravitasi bingkisan, sehingga:

T1 = T2 = mg

T1 = T2 = 100 N

Tegangan tali 3 menahan kedua tali 1 dan 2, sehingga:

T3 = T1 + T2

T3 = 100 N + 100 N

T3 = 200 N

Jadi, tegangan tali 1, tegangan tali 2 dan tegangan tali 3 berturut-turut adalah 100 N, 100 N dan 200 N.

Penjelasan:

Tegangan tali merupakan gaya yang terjadi pada tali akibat adanya tarikan dari dua benda yang saling terhubung dengan tali tersebut. Tegangan tali selalu mengarah ke arah tarikan benda, dan besarnya tegangan tali sama dengan besarnya gaya tarikan benda tersebut.

Dalam kasus ini, bingkisan seberat 100 N digantung di dengan dua tali. Tali 1 dan 2 masing-masing menahan gaya gravitasi bingkisan, sehingga tegangan tali 1 dan 2 sama dengan 100 N.

Tegangan tali 3 menahan kedua tali 1 dan 2, sehingga tegangan tali 3 sama dengan tegangan tali 1 + tegangan tali 2, yaitu 100 N + 100 N = 200 N.

33. perhatikan gambar dibawah jika gesekan katrol diabaikan maka besar tegangan tali t pada keadaan setimbang adalah


Untuk menyelesaikan soal ini kamu dapat menggunakan rumus-rumus berikut; dan untuk lebih memahaminya perhatikan gambar terlampir.

T cos θ = 12 N

T sin θ = 9 N

Maka,

tan θ = T sin θ / T cos θ, masukkan nilai-nilai yang diketahui

tan θ = 9 N / 12 N

tan θ = 3 / 4

tan θ = 0,75

θ = 37°

Maka, dengan

T cos θ = 12 N, yang kita cari adalah nilai tegangan tali (T), sehingga

T = 12 N / cos θ

T = 12 N / cos 37°

T = 12 N / 0,8

T = 15 N

Jadi, nilai tegangan tali pada sistem setimbang pada gambar adalah 15 N (D).

37. Perhatikan gambar kesetimbangan di bawah ini, jika massa bensa 8 kg maka besar gaya tegang tali T2 adalah


Jawaban:

T₂ = 64 N

Penjelasan:

Diketahui :

m = 8 kg => w = 8 kg × 10 m/s² = 80 N

θ₁ = 143°

θ₂ = 127°

θ₃ = 90°

Ditanya :

T₂

Jawab :

Sistem Ini dalam keadaan seimbang, jadi berlaku hubungan tegangan tali sebagai berikut :

[tex] \frac{T_{1} }{ \sin \theta_{1} } = \frac{T_{2} }{ \sin \theta_{2} } = \frac{W}{ \sin \theta_{3} } [/tex]

Karena yang ditanya hanya T₂ , maka kita gunakan :

[tex] \: \: \: \: \: \frac{T_{2} }{ \sin \theta_{2} } = \frac{W}{ \sin \theta_{3} } [/tex]

[tex] \frac{T_{2} }{ \sin (127^{o} ) } = \frac{80 \: N}{ \sin (90^{o} ) } [/tex]

[tex] \: \frac{T_{2} }{ \cos (37^{o} ) } = \frac{80 \: N}{ \sin (90^{o} ) } [/tex]

[tex] \: \: \: \: \: \: \: \: \frac{T_{2} }{ 0.8 } = \frac{80 \: N}{ 1 } [/tex]

[tex] \: \: \: \: \: \: \: \: T_{2} = 0.8 \times 80 \: N[/tex]

[tex] \: \: \: \: \: \: \: \: T_{2} = 64 \: N[/tex]

Jadj, Teganhan Tali T₂ = 64 N
