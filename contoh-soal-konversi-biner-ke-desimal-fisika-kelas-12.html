15. 1. Bilangan biner 10110011 di konversi ke bilangan desimal?2. Bilangan desimal 357 di konversi ke bilangan biner?3. Bilangan hexadesimal F9A2 konversi ke bilangan biner?​


Jawaban:

Konversi bilangan biner 1101011 ke dalam bilangan desimal dan hexadesimal adalah bilangan desimalnya 107 dan bilangan heksadesimalnya 6B. Sistem bilangan terdir dari 4 jenis yaitu bilangan biner (berbasis dua), bilangan oktal (berbasis delapan), bilangan desimal (berbasis sepuluh) dan bilangan heksadesimal (berbasis enam belas). Sistem bilangan yang sering digunakan adalah bilangan desimal, yaitu bilangan yang terdir dari angka 0, 1, 2, 3, 4, 5, 6, 7, 8 dan 9.

Pembahasan

Mengkonversi bilangan biner 1101011 ke dalam bilangan desimal

Bilangan biner tersebut terdiri dari 7 angka, sehingga kita mulai dari 2 pangkat 6

Jadi

Bilangan desimal dari bilangan biner 1101011 adalah

= 1 1 0 1 0 1 1

= (1 × 2⁶) + (1 × 2⁵) + (0 × 2⁴) + (1 × 2³) + (0 × 2²) + (1 × 2¹) + (1 × 2⁰)

= (1 × 64) + (1 × 32) + (0 × 16) + (1 × 8) + (0 × 4) + (1 × 2) + (1 × 1)

= 64 + 32 + 0 + 8 + 0 + 2 + 1

= 107

Mengkonversi bilangan biner 1101011 ke dalam bilangan heksadesimal

Karena bilangan biner tersebut terdiri dari 7 angka, maka agar bisa dikelompokkan menjadi 4 angka 4 angka maka didepan bilangan biner kita tambahkan angka 0,

Jadi

Bilangan desimal dari bilangan biner 1101011 adalah

= 1101011

= 01101011

= (0110) (1011)

= (0 × 2³ + 1 × 2² + 1 × 2¹ + 0 × 2⁰) (1 × 2³ + 0 × 2² + 1 × 2¹ + 1 × 2⁰)

= (0 × 8 + 1 × 4 + 1 × 2 + 0 × 1) (1 × 8 + 0 × 4 + 1 × 2 + 1 × 1)

= (0 + 4 + 2 + 0) (8 + 0 + 2 + 1)

= (6) (11)

= 6 B

= 6B

cara 1:

Tentu, saya akan membantu Anda dengan konversi bilangan tersebut.

1. Konversi bilangan desimal ke biner (Decimal to Binary):

  Bilangan desimal N = 1353

  Langkah-langkah:

  1. Bagi bilangan desimal N dengan 2.

  2. Catat sisa pembagian sebagai digit biner.

  3. Bagi hasil pembagian dengan 2 lagi, dan ulangi langkah 2, hingga hasil pembagian menjadi 0.

  Mari kita lakukan konversi ini:

  1353 / 2 = 676 sisa 1

  676 / 2 = 338 sisa 0

  338 / 2 = 169 sisa 0

  169 / 2 = 84 sisa 1

  84 / 2 = 42 sisa 0

  42 / 2 = 21 sisa 0

  21 / 2 = 10 sisa 1

2. Konversi bilangan desimal ke oktal (Decimal to Octal):

  Bilangan desimal N = 4355

  Langkah-langkah:

  1. Bagi bilangan desimal N dengan 8.

  2. Catat sisa pembagian sebagai digit oktal.

  3. Bagi hasil pembagian dengan 8 lagi, dan ulangi langkah 2, hingga hasil pembagian menjadi 0.

  Jadi, bilangan oktal untuk 4355 adalah 10343.

3. Konversi bilangan biner ke desimal (Binary to Decimal):

  Bilangan biner N = 1000101010

  Langkah-langkah:

  1. Tulis bilangan biner.

  2. Mulai dari digit paling kanan, kalikan setiap digit dengan 2 pangkat posisinya (dimulai dari pangkat 0) dan jumlahkan.

  = 554

  Jadi, bilangan desimal untuk 1000101010 adalah 554.

4. Konversi bilangan oktal ke desimal (Octal to Decimal):

  2. Mulai dari digit paling kanan, kalikan setiap digit dengan 8 pangkat posisinya (dimulai dari pangkat 0) dan jumlahkan.

  Mari kita lakukan konversi ini:

  5 * 8^0 + 4 * 8^1 + 2 * 8^2 + 6 * 8^3
