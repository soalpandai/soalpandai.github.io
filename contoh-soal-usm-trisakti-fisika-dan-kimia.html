8. Tolong dibantu kak.. Usm stis


∫dx/(e^x + 1)

misal, e^x + 1 = u
e^x = u - 1
e^x dx = du
dx = du/(e^x)
dx = du/(u - 1)

∫dx/(e^x + 1) = ∫(du/(u - 1))/u
= ∫du/(u(u - 1))
= ln|u - 1| - ln|u| + C
= ln |(u - 1)/u| + C
= ln|e^x/(e^x + 1)| + C
= ln|(e^x + 1)/(e^x)|^(-1) + C
= -ln|(e^x + 1)/(e^x)| + C
= -ln|1 + 1/e^x| + C
= -ln|1 + e^(-x)| + C

Jawaban tidak ada dalam opsi.

Jawaban:

Elang Mulia Lesmana (Fakultas Teknik Arsitektur, angkatan 1996)Hafidin Royan (Fakultas Teknik Sipil, angkatan 1996)Hendriawan Sie (Fakultas ekonomi, angkatan 1996)Hery Hartanto (Fakultas Teknik Mesin, angkatan 1995)

Tragedi Trisakti merupakan salah satu peristiwa pada masa orde baru, suatu peristiwa penembakan terhadap mahasiswa yang berdemonstrasi untuk menuntut turunnya Presiden Soeharto dari jabatannya, yang terjadi pada tanggal 12 Mei 1998. Korban tewas sebagai dampak ini sebanyak 4 orang mahasiswa. Empat mahasiswa tersebut dikenang sebagai pahlawan reformasi.

Latar belakang terjadinya tragedi Trisakti berawal dari latar belakang orde Baru secara ekonomi tepatnya sejak kondisi ekonomi Indonesia yang goyah pada tahun 1998. Kondisi ekonomi memburuk sebagai imbas dari krisis keuangan yang dialami Asia sejak tahun 1997 sampai 1999. Ketika itu banyak mahasiswa melakukan demonstrasi besar-besaran ke Gedung Nusantara termasuk mahasiswa Universitas Trisakti.
