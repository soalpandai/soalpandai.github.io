10. apa yang dimaksud DOC

Pada 1980an, WordPerfect menggunakan DOC sebagai format bawaan mereka. Kemudian, pada dasawarsa 1990an, Microsoftmemilih untuk menggunakan ekstensi DOC untuk format bawaaan Microsoft Word.

36. tentukan <doc sudut lancip Soalnya ada di lampiran TOLONG BANTU KAKAK MEMAKAI CARA ​


Jawab:

Penjelasan dengan langkah-langkah:

Karena ∠AOD, ∠DOC dan ∠COB ketiganya berpelurus maka jumlah ketiga sudut tersebut adalah 180°.

63 + ∠DOC + 42 = 180

∠DOC + 105 = 180

∠DOC = 180 - 105

∠DOC = 75°
