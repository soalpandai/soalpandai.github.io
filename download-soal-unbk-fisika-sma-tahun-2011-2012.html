36. Diketahui indeks harga konsumen tahun 2011 sbesar 147,50 % dan indeks harga tahun 2012 sebesar 149,25 %, maka laju inflasi tahun 2012 adalah...â€‹


Jawaban:

Laju inflasi tahun 2012 adalah sebesar 1,1864% atau sekitar 1,19%.

Penjelasan:

Untuk menghitung laju inflasi antara tahun 2011 dan tahun 2012, kita dapat menggunakan rumus sebagai berikut:

Laju Inflasi = ((Indeks Harga Tahun 2012 - Indeks Harga Tahun 2011) / Indeks Harga Tahun 2011) * 100

Dalam kasus ini, indeks harga konsumen tahun 2011 adalah 147,50% dan indeks harga konsumen tahun 2012 adalah 149,25%.

Laju Inflasi = ((149,25 - 147,50) / 147,50) * 100

Laju Inflasi = (1,75 / 147,50) * 100

Laju Inflasi = 0,011864 * 100

Laju Inflasi = 1,1864

Jadi, laju inflasi tahun 2012 adalah sebesar 1,1864% atau sekitar 1,19%.
