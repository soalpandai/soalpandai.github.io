Jawaban:

0,8 m

Penjelasan:

Diketahui :

k1 = 100 N/m

k2 = 150 N/m

W = 50 N

Ditanyakan : Δx ?

Dijawab :

1/k tot = 1/k1 + 1/k2

= 1/100 + 1/150

= (3 + 2)/300

= 5/300

= 300/5

= 60

Jadi pertambahan panjang :

W = F = ktot Δx

Δx = W/k tot

= 50/60

= 0,8

jadi pertambahan panjang pegas adalah 0,8 m

Grade : X SHS

Jawaban:

C. 20,7 m/s

Penjelasan:

Given :

m1 = 1000 kgv1 = 25.0 m/sm2 = 9000 kgv2 = 20.0 m/sv1' = 18.0 m/s

Solving Way:

Using formula of momentum,

m1v1 + m2v2 = m1v1' + m2v2'

(1000×25) + (9000×20) = (1000×18) + (9000×v2')

25 000 + 180 000 = 18 000 + 9000v2'

9000v2' = 187 000

v2' = 187000 / 9000

v2' = 20,7

So, the velocitiy of the truck right after the collition is (C) 20,7 m/s.

18. ( BIG POINT )MAPEL : FISIKAKELAS : 10 SMAJAWAB BESERTA PENJELASANJANGAN NGASAL-@@-​


diket:

rA = 10 cm = 0,1 m

rB = 5 cm = 0,05 m

rC = 15 cm = 0,15 m

ωA = 2 rad/s

ditanya:

a. kelajuan linier roda A (vA)....?

b. kecepatan sudut roda B (ωB)...?

c. kelajuan linier roda B (vB)...?

d. kelajuan linier roda C (vC)...?

e. kecepatan sudut roda C (ωC)....?

jawab:

a. kelajuan linier roda A (vA)

   vA = rA ωA

   vA = 0,1 m x 2 rad/s

        = 0,2 m/s

Jadi, kelajuan linier roda A adalah 0,2 m/s.

b. kecepatan sudut roda B (ωB)

   karena roda A dan roda B saling menempel, maka

   ωA = ωB

   ωB = 2 rad/s

Jadi, kecepatan sudut roda B adalah 2 rad/s.

c. kelajuan linier roda B (vB)

   vB = rB ωB

   vB = 0,05 m x 2 rad/s

        = 0,1 m/s

Jadi, kelajuan linier roda B adalah 0,1 m/s.

d. kelajuan linier roda C (vC)

   karena roda B dan roda C terhubung rantai, maka

   vB = vC

   vC = 0,1 m/s

Jadi, kelajuan linier roda C adalah 0,1 m/s.

e. kecepatan sudut roda C (ωC)

   ωC = vC / rC

   ωC = 0,1 m/s : 0,15 m

          = 0,67 rad/s

Jadi, kecepatan sudut roda C adalah 0,67 rad/s.
