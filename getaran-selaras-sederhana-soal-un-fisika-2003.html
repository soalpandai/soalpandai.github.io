PEMBAHASAN

Marilah kita ingat kembali persamaan simpangan getaran selaras berikut ini:

[tex]\boxed{y = A \sin \theta = A \sin \omega t}[/tex]

dimana :

y = simpangan getaran ( meter )

A = simpangan maksimum ( meter )

θ = sudut fase ( radian )

ω = frekuensi sudut ( rad/s )

t = selang waktu ( sekon )

Mari kita sekarang mencoba menyelesaikan soalnya.

[tex]\texttt{ }[/tex]

Amplitudo = A = 8 cm

Fase getaran = φ = ¹/₁₂

Ditanyakan:

Simpangan getaran = y = ?

Penyelesaian:

[tex]y = A \sin \theta[/tex]

[tex]y = A \sin (2 \pi \varphi)[/tex]

[tex]y = A \sin (2 \pi \times \frac{1}{12})[/tex]

[tex]y = A \sin (\frac{1}{6} \pi)[/tex]

[tex]y = 8 \sin (30^o)[/tex]

[tex]y = 8 \times \frac{1}{2}[/tex]

[tex]\boxed{y = 4 \texttt{ cm}}[/tex]

10. Sebuah bandul matematis bergetar selaras sederhana pada pegas dengan tetapan gaya 80 N/m. Jika massa bandul 0,2 kg dan amplitudo getaran 20 cm,kecepatan maksimum bandul sebesar .........


JAWABAN :

Berdasarkan soal diatas diketahui bahwa :
- tetapan gaya adalah 80 N/m -> k = 80 N/m
- massa bandul adalah 0,2 kg -> m = 0,2 kg
- amplitudo getaran adalah 20 cm -> A = 20 cm
Selanjutnya, karena yang ditanyakan adalah kecepatan maksimum bandul, maka perlu mencari Vmax pada bandul.
Untuk mencari kecepatan maksimum bandul caranya sebagai berikut :
V max = A. ω
Tapi mula - mula kita mencari ω dulu sebagai berikut :
k = m. ω2
ω = √k/m
ω = √80/0,2
ω = √400
ω = 20rad/s
Sekarang baru mencari kecepatan maksimumnya sebagai berikut :
V max = A. ω
V max = 20 cm . 20 rad/s
V max = 400 cm/s
V max = 4 m/s
Berdasarkan uraian diatas dapat disimpulkan bahwa besarnya kecepatan maksimum pada bandul adalah 4 m/s.


PEMBAHASAN LEBIH LANJUT :

Persamaan untuk mencari kecepatan maksimum adalah sebagai berikut :
V max = A. ω
Keterangan :
V max = kecepatan maksimum (satuan m/s)
A = Amplitudo
ω = frekuensi sudut
Adapun persamaan pada ω sebagai berikut :
ω = √k/m
Keterangan :
ω = frekuensi sudut
k = konstanta atau tetapan bandul (satuan N/m)
m = massa benda (satuan kg)

21. Periode getaran selaras yang terjadi pada sebuah bandul sederhana pada waktu siang hari adalah ts dan pada waktu malam hari yang dingin tm,jika dibandingkan,maka.


Jawaban:

Nilai periode getaran suatu bandul sederhana di Bumi akan lebih kecil di bandingkan dengan nilai periode getaran di Bulan.

Pembahasan

Getaran Harmonis Sederhana adalah gerakan periodik dimana besarnya percepatan getaran berbanding lurus dengan simpangan getaran tetapi dalam arah yang berbeda.

a = -ω². x

dimana :

a = percepatan getaran ( m/s² )

ω = frekuensi sudut ( rad )

x = simpangan getaran ( m )

Salah satu contoh getaran harmonis sederhana adalah benda yang digantung dengan tali dan di beri simpangan kecil sehingga mengalami getaran atau disebut juga dengan nama bandul.

[tex]T = 2\pi \sqrt{\frac{L}{g}}[/tex]

dimana :

T = periode bandul ( s )

L = panjang bandul ( m )

g = percepatan gravitasi ( m/s² )

Pegas yang digetarkan juga merupakan contoh dari gerak harmonik sederhana , untuk mencari periode dari getaran pegas ini bisa memakai rumus berikut ini :

[tex]T = 2\pi \sqrt{ \frac{m}{k}}[/tex]

dimana :

T = periode pegas ( s )

m = massa beban pada pegas ( m )

k = konstanta pegas ( m/s² )

Simpangan yang di lakukan oleh benda yang bergetar secara harmonis bisa di rumuskan sebagai berikut ini :

y = A . sin ωt

dimana :

y = simpangan getaran ( m )

A = amplitudo getaran ( m )

ω = frekuensi sudut ( rad/s )

t = waktu getaran ( s )

Untuk mendapatkan kecepatan getaran bisa di turunkan dari rumus simpangan , yakni :

v = dy/dt

v = Aω cos ωt

dimana :

v = kecepatan getaran ( m )

A = amplitudo getaran ( m )

ω = frekuensi sudut ( rad/s )

t = waktu getaran ( s )

Percepatan getaran bisa di cari dengan cara menurunkan persamaan dari kecepatan getaran menjadi :

a = dv/dt

a = -Aω² sin ωt

dimana :

v = kecepatan getaran ( m )

A = amplitudo getaran ( m )

ω = frekuensi sudut ( rad/s )

t = waktu getaran ( s )

Oke, mari kita gunakan rumus ini untuk menyelesaikan soal tersebut.

Bulan adalah satu-satunya satelit alami dari Bumi. Percepatan gravitasi bulan sekitar ⅙ kali dari percepatan gravitasi di Bumi. Perbedaan percepatan gravitasi ini mempengaruhi gerak benda di Bumi dan di Bulan.

Periode getaran bandul sederhana di pengaruhi oleh panjang bandul dan percepatan gravitasi sesuai rumus :

[tex]T = 2\pi \sqrt{ \frac{L}{g}}[/tex]

dimana :

T = periode bandul ( s )

L = panjang bandul ( m )

g = percepatan gravitasi ( m/s² )

Pada rumus ini terlihat jika periode bandul sederhana berbanding terbalik dengan akar besarnya percepatan gravitasi tempat dimana percobaan di lakukan.

Kesimpulan

Nilai periode getaran suatu bandul sederhana di Bumi akan lebih kecil di bandingkan dengan nilai periode getaran di Bulan.

ω = frekuensi sudut ( rad/s )

t = waktu getaran ( s )

Oke, mari kita gunakan rumus ini untuk menyelesaikan soal tersebut.

Bulan adalah satu-satunya satelit alami dari Bumi. Percepatan gravitasi bulan sekitar ⅙ kali dari percepatan gravitasi di Bumi. Perbedaan percepatan gravitasi ini mempengaruhi gerak benda di Bumi dan di Bulan.

Periode getaran bandul sederhana di pengaruhi oleh panjang bandul dan percepatan gravitasi sesuai rumus :

[tex]T = = 2\pi \sqrt{ \frac{L}{g}}[/tex]

dimana :

T = periode bandul ( s )

L = panjang bandul ( m )

g = percepatan gravitasi ( m/s² )

Pada rumus ini terlihat jika periode bandul sederhana berbanding terbalik dengan akar besarnya percepatan gravitasi tempat dimana percobaan di lakukan.

Kesimpulan

Nilai periode getaran suatu bandul sederhana di Bumi akan lebih kecil di bandingkan dengan nilai periode getaran di Bulan.

39. sebuah bandul matematis bergetar selaras sederhana pada pegas dengan tetapan gaya 80n/m. jika massa bandul 0,2 kg dan amplitudo getaran 20 cm,kecepatan maksimum bandul sebesar


JAWABAN :

 

Dari soal dapat diketahui bahwa:

- tetapan gaya > k = 80 N/m
 - massa bandul > m = 0,2 kg 
 - amplitudo getaran > 20 cm

 

Ditanyakan kecepatan maksimum > v max  

Untuk mencari kecepatan maksimum bandul, rumusnya adalah : 

V max = A. ω

 

Mula – mula kita cari ω dahulu:

 

k = m[tex] w^{2} [/tex]
ω = [tex] \sqrt{ \frac{k}{m} } \\ = \sqrt{ \frac{80}{0,2} } \\ = \sqrt{400} \\ = 20 rad/s[/tex]

 

sehingga kecepatan maksimumnya,

 

V max = A. ω

V = 20. 20 = 400 cm/s = 4 m/s.

 

Berdasarkan uraian di atas, didapat bahwa kecepatan maksimum sebesar 400 cm/s = 4 m/s.

 

 

PEMBAHASAN LEBIH LANJUT :

 

Persamaan simpangan gerak harmonis yaitu:

Y = A sin (ωt)

 

Persamaan kecepatan gerak harmonis yaitu:

V = Aω cos (ωt)

 

Persamaan percepatan gerak harmonis yaitu:

a = A ω^2 sin (ωt)

 

Keterangan:

Y = simpangan

A = amplitudo

ω = frekuensi sudut

t = waktu

v = kecepatan

a = percepatan
