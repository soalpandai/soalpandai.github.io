Nilai dari [tex]\frac{2015^2(2014^2-2013)}{(2014^2-1)(2014^3+1)}\times\frac{2013^2(2014^2+2015)}{(2014^3-1)}[/tex] adalah 1.

PEMBAHASAN

Pangkat atau eksponen adalah perkalian berulang dari suatu bilangan yang sama. Bentuk [tex]a^b[/tex] berarti kita mengalikan bilangan a dengan bilangan a sebanyak b kali atau

[tex]a^b=\underbrace{a\times a\times a\times...\times a}_{sebanyak~b~kali}[/tex]

Operasi pada bilangan pangkat adalah sebagai berikut.

[tex](i)~a^b\times a^c=a^{b+c}[/tex]

[tex](ii)~\frac{a^b}{a^c}=a^{b-c}[/tex]

[tex](iii)~(a^b)^c=a^{b\times c}[/tex]

[tex](iv)~a^{-b}=\frac{1}{a^b}[/tex]

[tex](v)~\sqrt[n]{a}=a^{\frac{1}{n}}[/tex]

DIKETAHUI

[tex]\frac{2015^2(2014^2-2013)}{(2014^2-1)(2014^3+1)}\times\frac{2013^2(2014^2+2015)}{(2014^3-1)}=[/tex]

.

DITANYA

Tentukan hasilnya.

.

PENYELESAIAN

Misal :

2014 = a

2013 = a-1

2015 = a+1

.

[tex]\frac{2015^2(2014^2-2013)}{(2014^2-1)(2014^3+1)}\times\frac{2013^2(2014^2+2015)}{(2014^3-1)}[/tex]

[tex]=\frac{(a+1)^2(a^2-a+1)}{(a^2-1)(a^3+1)}\times\frac{(a-1)^2(a^2+a+1)}{(a^3-1)}[/tex]

[tex]=\frac{(a+1)^2(a^2-a+1)}{(a+1)(a-1)(a^3+1)}\times\frac{(a-1)^2(a^2+a+1)}{(a^3-1)}[/tex]

[tex]=\frac{(a+1)(a^2-a+1)}{(a^3+1)}\times\frac{(a-1)(a^2+a+1)}{(a^3-1)}[/tex]

[tex]=\frac{(a^3-a^2+a+a^2-a+1)}{(a^3+1)}\times\frac{(a^3+a^2+a-a^2-a-1)}{(a^3-1)}[/tex]

[tex]=\frac{(a^3+1)}{(a^3+1)}\times\frac{(a^3-1)}{(a^3-1)}[/tex]

[tex]=1\times1[/tex]

[tex]=1[/tex]

A. Mencari nilai [OH-]
--> Soal UN ini mengaitkan 2 bab (kyk SBMPTN, ya), yaitu hidrolisis garam dan hasil kali kelarutan (Ksp), sehingga:
     CH3COOH + NaOH --> CH3COONa + H2O

b        -0.8          -0.8              +0.8          +0.8
s           -              -                 0.8

[G] = 0.8 mmol / 200 mL = 4 x 10^-3

Bersisa garamnya sebesar 0,8 mmol, masukkan ke rumus hidrolisis:
[OH-] = √Kw x [G]/Ka
         = √10^-14 x 4 x 10^-3 / 10^-5
         = 2 x 10^-6

B. Mencari [Fe2+] dari Fe(OH)2
--> Untuk mencarinya, buat reaksi ionisasi Besi (II) hidroksida tsb:
Fe(OH)2 ⇄ Fe2+ + 2OH-

1,6 x 10^-15 = [Fe2+] [2 x 10^-6]^2

           = 0.4 x 10^-3

= 3 × 300 - 3

= 900 - 3

21. SBMPTN 2014 bidang FISIKA.


Oke disini ada asumsi bahwa :
- pada ujung dasar tabung selalu terdapat simpul (bila dihitung dari bawah, gelombang dimulai dengan simpul)
- perut gelombang terjadi pada jarak Ln + ΔL dari dasar tabung

untuk resonasi ke n, dapat dicari panjang seluruh gelombang pada tabung adalah
λn = (1/4 + (1/2(n-1))λ
mis. pada resonasi pertama n = 1 maka panjang seluruh gelombang adalah 1/4λ dihitung dari dasar tabung, dan bertambah dgn kelipatan setengah

kita misalkan total gelombang pada tabung adalah λn dengan jarak tempuhnya adalah Ln + ΔL
Ln + ΔL = λn
λn = (1/4 + (1/2(n-1))λ
Ln + ΔL = (1/4 + (1/2(n-1))λ
kalikan kedua sisi dgn 4
4 (Ln + ΔL) = (1 + 2n - 2)λ
4 (Ln + ΔL) = (2n - 1)λ
maka kita dapat
λ = (4(Ln + ΔL)) / (2n - 1)

ingat v = λf
maka f = v/λ
f = v / ((4(Ln + ΔL)) / (2n - 1))
f = v (2n - 1) / (4(Ln + ΔL))
