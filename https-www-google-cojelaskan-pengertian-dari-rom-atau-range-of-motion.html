15. Apa pengertian dari ROM​


Read Only Memory atau dapat disingkat dengan ROM yang mempunyai kemampuan untuk menyimpan data secara permanen. Ketika sumber dayanya mati, ROM tidak akan kehilangan datanya, lantaran dijalankan serta ditopang oleh baterai kecil yang terpasang pada sebuah komputer. ROM ini memiliki fungsi utama untuk input dan output data, serta memegang kendali atas program perangkat lunak.  

Pembahasan

Biasanya ROM berbentuk chip serta terletak di motherboard dan papan sirkuit. Tujuannya adalah untuk mengatur input dan output dari sistem manajemen data dasar, serta membaca dan menulis ke perangkat.  

ROM ini menyediakan instruksi yang diperlukan tentang cara perangkat saling terhubung dengan komponen lain. ROM sendiri biasanya digunakan sebagai firmware. Firmware merupakan sekumpulan program yang ditanamkan ke dalam mekanisme sebuah perangkat keras komputer atau hardware. ROM umumnya dapat dijumpai di antara media penyimpanan optik, seperti CD-ROM, CD-R atau singkatan dari Compact Disc Recordable, dan CD-RW atau singkatan dari Compact Disc Re-Writeable.  

Secara sederhana, ROM dapat disebut sebagai tempat untuk menyimpan data internal yang juga bisa untuk menjalankan sistem operasi serta sistem yang lainnya.

ROM ini penyimpanannya permanen, Non-volatile atau tidak mudah 'menguap', kemudian digunakan untuk proses startup komputer, serta menulis datanya lebih lambat.

16. 1. Jelaskan pengertian Animasi motion guide? 2. Jelaskan langkah-langkah dalam membuat animasi motion guide?


Jawaban:

1) Animasi motion guide adalah animasi yang mempunyai gerakan sesuai dengan jalur yang kita buat. Animasi ini merupakan lanjutan dari animasi motion tween. Animasi ini sangat cocok digunakan untuk jenis animasi yang membutuhkan ketelitian dalam pergerakan yang dikehendaki atau sesuai keinginan pem-buat animasi.

2) langkah-langkahnya yaitu :

1. Langkah pertama untuk membuat animasi motion guide, buatlah lingkaran dengan menggunakan ovaltool. buat bentuk oval menggunakan oval tool copy

2. Lalu klik icon selection tool, klik kanan pada frame ke 20 dan pilih insert keyframe. lalu klik kanan pada frame ke 20 dan pilih insert key frame copy

3. Setelah itu klik add motion guide, lalu pilih add motion guide copy

4. Setelah klik add motion guide lalu klik frame pertama pada layer guide, lalu pilih frame pertama pada guide copy

5. Lalu ganti stroke color menjadi warna hitam dan fill color menjadi tidak ada warna, lalu ganti stroke color menjadi warna hitam copy, dan fill color menjadi tidak ada warna copy

6. setelah di ganti warna, buatlah bentuk oval dengan menggunakan ovaltool, lalu buat oval menggunakan oval tool copy

7. Kemudian klik icon selection tool, lalu blok pada bagian ujung oval (hanya sedikit), lalu di blok copy

8. Setelah ujung oval tersebut di blok lalu hapus dan klik layer satu pada frame pertama, lalu hapus dan klik layer1 pada frame pertama copy

9. Selanjutnya pindahkan lingkaran seperti gambar dibawah ini, dan pindahkan lingkaran seperti gambar di bawah ini copy

11. Selanjutnya klik kanan pada frame layer1 dan klik create motion tween, selanjutnya klik kanan pada frame layer1 dan klik create motion tween copy

23. Pengertian dari www adalah?


WWW atau World Wide Web adalah suatu program yang ditemukan oleh Tim Berners-Lee pada tahun 1991. Awalnya Berners-Lee hanya ingin menemukan cara untuk menyusun arsip-arsip risetnya. Untuk itu, dia mengembangkan suatu sistem untuk keperluan pribadi. Sistem itu adalah program peranti lunak yang diberi nama Equire. Dengan program itu, Berners-Lee berhasil menciptakan jaringan terkait antara berbagai arsip sehingga memudahkan informasi yang dibutuhkan. Inilah yang kemudian menjadi dasar dari sebuah revolusi yang dikenal sebagai web.
WWW dikembangkan pertama kali di Pusat Penelitian Fisika Partikel Eropa (CERN), Jenewa, Swiss. Pada tahun 1989 Berners-lee membuat proposal untuk proyek pembuatan hypertext secara global, kemudian pada bulan Oktober 1990, ‘World Wide Web‘ sudah bisa dijalankan dalam lingkungan CERN. Pada musim panas tahun 1991, WWW resmi digunakan secara luas pada jaringan Internet.

34. apakah perbedaan www dan https?​


HTTPS atau hyper-text transfer protocol secure adalah serangkaian peraturan dalam komunikasi antara 2 system seperti komputer client dan web server. HTTPS menambahkan protocol secure untuk mengamankan transfer hyper-text 2 system tersebut dengan cara enkripsi dan dekripsi. Sedangkan WWW atau World Wide Web adalah prefix dari domain name system atau DNS untuk menyatakan bahwa website yang diakses menggunakan protocol HTTP/HTTPS dan dapat diakses melalui internet. WWW dapat digunakan sebagai penanda bahwa URL yang diakses adalah HTTP/HTTPS. Hal ini dapat membantu untuk membedakan file server yang menggunakan FTP dengan web server yang menggunakan HTTP/HTTPS sebagai transfer protocolnya.
