[tex]F=k\frac{Q_{1} Q_{2}}{r^{2} }[/tex]

dimana Q₁ = Q₂ = Q dan k = 9 x 10⁹ Nm²/C², sehingga :

[tex]F=k\frac{Q^{2} }{r^{2} }[/tex]

[tex]0,9N=9x10^{9} Nm^{2} /C^{2} \frac{Q^{2} }{(0,5m)^{2} }[/tex]

(0,5m)² 0,9 N = 9 x 10⁹ Nm²/C² Q²

0,25 m² 0,9 N = 9 x 10⁹ Nm²/C² Q²

0,225 m²N = 9 x 10⁹ Nm²/C² Q²

Q² = 0,225 m²N / 9 x 10⁹ Nm²/C²

Q² = 0,025 x 10⁻⁹ C²

Q² = 0,25 x 10⁻¹⁰ C²

Q = 0,5 x 10⁻⁵ C

Q = 5 x 10⁻⁶ C

Q = 5μC

[tex]\boxed{Kesimpulan}[/tex]

Besar muatan tersebut adalah 5μC.
