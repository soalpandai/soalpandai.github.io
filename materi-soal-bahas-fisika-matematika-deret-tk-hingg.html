29. Kuis MatematikaMateri : Harga MutlakTk. olimpiade> Dilarang Spam> Jawab dengan Langkah Sejelas-jelasnya


x/(|x|) + (|y|)/y = 2y

*x,y ≠ 0*

(i)
Untuk x > 0
x/x + (|y|)/y = 2y
y + |y| = 2y²
|y| = 2y² - y

(i.a)
Jika y > 0
y = 2y² - y
0 = 2y² - 2y
y = 0 V y = 1

(i.b)
Jika y < 0
-y = 2y² - y
0 = 2y²
y = 0

(ii)
Untuk x < 0
-x/x + (|y|)/y = 2y
-y + |y| = 2y²
|y| = 2y² + y

(ii.a)
Jika y > 0
y = 2y² + y
0 = 2y²
y = 0

(ii.b)
Jika y < 0
-y = 2y² + y
0 = 2y² + 2y
y = 0 V y = -1


Dari kemungkinan di atas, y = 0 tidak memenuhi sehingga daerah hasilnya adalah {-1, 1}

2. Berikan isiannya gula merah jawa yang halus dan bentuk seperti bola kecil.
