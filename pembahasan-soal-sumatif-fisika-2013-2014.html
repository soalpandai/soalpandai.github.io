angka satuan pada bilangan 2013ⁿ

= angka satuan pada bilangan 3ⁿ

2013² => 3² = 9

2013³ => 3³ = 27

2013⁴ => 3⁴ = 81

2013⁵ => 3⁵ = 243

2013⁶ => 3⁶ = 729

2013⁷ => 3⁷ = 2.187

2013⁹ => 3⁹ = 19.683

dan seterusnya

(tidak mungkin kita teruskan sampai pangkat 2015)

Sudat terlihat polanya, angka satuannya selalu berulang dari :

3, 9, 7, 1, 3, 9, 7, 1, dan seterusnya selalu berulang tiap 4 bilangan

Jadi untuk menentukan satuan pada 2013ⁿ yaitu dengan cara (n : 4)

Jika bersisa 0 => angka satuannya = 1

Jika bersisa 1 => angka satuannya = 3

Jika bersisa 2 => angka satuannya = 9

Jika bersisa 3 => angka satuannya = 7

Jadi

2015 : 4 = 503 bersisa 3 maka

angka satuannya adalah 7

kenapa 2015 : 4 = 503 bersisa 3

karena 503 × 4 = 2012 sehingga agar hasilnya 2015 => 2012 + 3

20. soal UN 2013/2014 no.13


maaf harus di poto :)
clik gambar biar liat penuh :)Diket dan dit:
*n NaOH = 100 mL x 0,008 M = 0.8 mmol
*n CH3COOH = 100 mL x 0,008 M = 0,8 mmol
*Ka = 10^-5
*Ksp Fe(OH)2 = 1,6 x 10^-15
*[Fe2+] = ? M

A. Mencari nilai [OH-]
--> Soal UN ini mengaitkan 2 bab (kyk SBMPTN, ya), yaitu hidrolisis garam dan hasil kali kelarutan (Ksp), sehingga:
     CH3COOH + NaOH --> CH3COONa + H2O
m        0.8           0,8
b        -0.8          -0.8              +0.8          +0.8
s           -              -                 0.8

[G] = 0.8 mmol / 200 mL = 4 x 10^-3

Bersisa garamnya sebesar 0,8 mmol, masukkan ke rumus hidrolisis:
[OH-] = √Kw x [G]/Ka
         = √10^-14 x 4 x 10^-3 / 10^-5
         = 2 x 10^-6

B. Mencari [Fe2+] dari Fe(OH)2
--> Untuk mencarinya, buat reaksi ionisasi Besi (II) hidroksida tsb:
Fe(OH)2 ⇄ Fe2+ + 2OH-

Ksp = [Fe2+] [OH-]^2
1,6 x 10^-15 = [Fe2+] [2 x 10^-6]^2
[Fe2+] = 1,6 x 10^-15 / 4 x 10^-12
           = 0.4 x 10^-3
           = 4 x 10^-4 Molar (D)
