38. Diketahui barisan aritmetika dengan Un adalah suku ke-n. Jika U2 + U15 + U40 = 165, maka U19 sama dengan ... (Ujian Nasional UN 2010)


Un = a + (n - 1)b

U₂ + U₁₅ + U₄₀ = 165
(a + b) + (a + 14b) + (a + 39b) = 165
3a + 54b = 165
a + 18b = 55

U₁₉ = a + (19 - 1)b
U₁₉ = a + 18b
U₁₉ = 55

Jadi, suku ke-19 adalah 55
