2. definisi internet dan web brouser


definisi internet adalah jaringan besar yang saling berhubungan dari jaringan-jaringan komputer yang menghubungkan orang-orang dan komputer-komputer diseluruh dunia, melalui telepon, satelit dan sistem-sistem komunikasi yang lain. Internet dibentuk oleh jutaan komputer yang terhubung bersama dari seluruh dunia, memberi jalan bagi informasi (mulai dari text, gambar, audio, video, dan lainnya ) untuk dapat dikirim dan dinikmati bersama. Untuk dapat bertukar informasi, digunakan protocol standar yaitu Transmision Control Protocol dan internet Protocol yang lebih dikenal sebagai TCP/IP.

web browser adalah untuk membawa sumber informasi kepada pengguna. Proses ini dimulai ketika pengguna memasukan sebuah Uniform Resource Identifier (URI), misalnya, http://anneahira.com ke dalam browser.

22. Pengertian dari Web 2.0 Dan tahun berapa Muncul Web 2.0?


Pengertian Web 2.0 :

Web 2.0 disebut sebagai suatu generasi kedua dari layanan berbasis web dalam dunia komputer yang dijalankan langsung diatas internet sebagai platformnya.

Tahun berapa munculnya web 2.0 :

Pada tahun 2003 Web 2.0 dipelopori oleh O'Reilly Media, kemudian sekitar tahun 2004, web 2.0 mulai dikembangkan dengan menyatukan teknologi-teknologi gabungan dari HTML, XML, CSS, JavaScript, dan AJAX. HTML dan CSS digunakan untuk mempercantik tampilan web. XML digunakan untuk mendefinisikan format suatu data.

23. pengertian pemogramam web?


Pemprograman web adalah suatu Kode Pemprograman untuk melakukan Tugas/Task otomatis yang di lakukan di suatu Website

Seperti
HTML, PHP, dan lain"Pemrograman web merupakan proses untuk menciptakan aplikasi dengan media internet yang disimpan dalam bentuk hosting atau media penyimpanan web. Pemrograman web berbeda dengan pemrograman desktop. Aplikasi yang dibuat dengan pemrograman web lebih bersifat universal dan bisa diakses hampir semua orang, karena tidak memerlukan proses instalasi seperti pada aplikasi yang dibuat di desktop.
Kelemahan aplikasi web yakni pada aksesbilitasnya, dimana koneksi internet akan sangat mempengaruhi kecepatan transfer data dan proses eksekusi program web. Selain itu, risiko terhadap hacking dan pencurian data lebih terasa pada aplikasi web.
Kelebihan aplikasi web yakni pada kompatibilitas dan cakupan pengguna yang luas.
Contoh Aplikasi Pemrograman Web: Website E-Commerce (toko online), Scribd (situs menyimpanan dokumen), Facebook (jejaring sosial), dll
