4. 1. Jelaskan pengertian informatika? 2. Apa pengertian dari database? 3. Jelaskan pengertian dari coding? 4. Apa yang dimaksud dengan kecerdasan buatan (AI) ? 5. Bagaimana cara kerja CPU pada sistem komputer? Jelaskan fungsi dari ALU dan CU pada CPU!


Jawaban:

1. Informatika adalah disiplin ilmu yang mempelajari pengolahan, pengelolaan, dan penggunaan informasi dengan bantuan komputer. Fokus utama informatika adalah memahami dan menerapkan prinsip-prinsip komputasi untuk memecahkan masalah dan mengelola data dengan efisien.

2. Database adalah kumpulan data yang terstruktur yang disimpan secara elektronik. Dalam database, data diatur dalam tabel atau entitas yang berhubungan satu sama lain, dan dapat diakses, dimodifikasi, atau diperbarui oleh pengguna atau aplikasi.

3. Coding (pemrograman) adalah proses membuat kode atau instruksi untuk memprogram komputer agar melakukan tugas atau fungsi tertentu. Pemrograman melibatkan penulisan kode-kode yang terstruktur menggunakan bahasa pemrograman yang dipahami oleh mesin, sehingga dapat dijalankan dan menghasilkan output yang diinginkan.

4. Kecerdasan Buatan (Artificial Intelligence/AI) mengacu pada kemampuan mesin atau komputer untuk meniru dan melakukan tugas yang biasanya membutuhkan kecerdasan manusia. AI mencakup berbagai teknik seperti machine learning, deep learning, dan neural networks untuk memungkinkan mesin belajar dari data dan mengambil keputusan atau melakukan tugas tertentu secara otomatis.

5. CPU (Central Processing Unit) adalah otak komputer yang bertanggung jawab untuk menjalankan instruksi dan melakukan operasi pada sistem komputer. CPU terdiri dari dua komponen utama: ALU (Arithmetic Logic Unit) dan CU (Control Unit).

- ALU bertanggung jawab untuk melakukan operasi matematika dan logika dasar, seperti penjumlahan, pengurangan, perkalian, dan perbandingan. ALU juga mengendalikan aliran data dalam CPU.

- CU bertanggung jawab untuk mengarahkan dan mengkoordinasikan operasi di dalam CPU. Ini mengambil instruksi dari memori, mengurai instruksi tersebut, mengarahkan aliran data dan sinyal kontrol dalam CPU, serta mengatur eksekusi instruksi secara berurutan. CU menjaga sinkronisasi dan koordinasi antara komponen CPU dan perangkat keras lainnya.

23. Dalam sebuah database memiliki beberapa tingkatan database, sebutkan dan jelaskan!


Jawaban:

Tingkatan data dalam database mulai dari yang terbesar ke yang terkecil yaitu :

1.      Database

Suatu database  menggambarkan data yang saling berhubungan antara satu dengan yang lainnya.

2.   File

     Suatu file menggambarkan suatu kesatuan data yang sejenis, dimana kumpulan dari file membentuk suatu database.

3.   Record

     Suatu record  menggambarkan suatu unit data individu yang tertentu dimana kumpulan dari record  membentuk suatu file.

4.   Field

     Suatu  field  menggambarkan suatu attribute dari record, dimana kumpulan field  membentuk suatu record.

5.   Byte

     Attribute dari  field  berupa huruf yang membentuk nilai dari sebuah field.

6.   Bit

           Merupakan bagian terkecil dari data secara keseluruhan yaitu berupa karakter ASCII (American Standar Code Form Information Intercharge). 0 (nol) adalah satu yang merupakan komponen pembentuk byte.

33. Apakah pengertian dari database, server, proxy, ssh, dan ip :)


database, menurut saya adalah sekumpulan data yang telah tersimpan dan diatur, sehingga bisa dicari atau diambil debngan efisien ...
server, sebuah sistem komputer yang menyediakan jenis layanan tertentu dalam sebuah jaringan komputer ( kurang paham )
proxy, suatu mekanisme dan service serta jembatan anda ke facebook.com sekaligus memberikan konten yang sudah/bisa di cache/save di direktory proxy server untuk saat nanti jika anda mengakses ke facebook.com lagi tanpa harus mengambil langsung dari server facebook. 
ssh, (penjelasanya susah, tapi saya sering gunain ssh untuk gratisan)
ip, nama komputer anda dalam nilai angka di jaringan, komputer mengenali komputer lainnya dengan angka misal 192.168.1.100 bukannya nama dalam huruf seperti hostname atau domain seperti facebook.com. 

database = kumpulan informasi yang disimpan di dalam komputer secara sistematik untuk memperoleh informasi dari basis data tersebut.
server = sebuah sistem komputer yang menyediakan jenis layanan tertentu dalam sebuah jaringan komputer.
proxy = suatu server yang menyediakan layanan untuk meneruskan setiap permintaan kita kepada server lain di internet.
ssh = (Secure Shell) protokol jaringan yang memungkinkan pertukaran data melalui saluran aman antara dua perangkat jaringan.
ip = (internet protocol) suatu cara untuk mengetahui asal atau alamat suatu komputer berupa sistem penomoran masing-masing komputer yang bersifat unik atau tidak sama. Sistem penomoran itu sendiri terdiri dari empat bagian yang dipisahkan oleh titik contoh : 202.155.245.2
