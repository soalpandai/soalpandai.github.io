11. Seorang sisawa SMK jurusan listrik sedang merangkai 8 resistor secara seri. Nilai hambatan setiap resistor adalah 2 ohm, 10 ohm, 50 ohm, dan seterusnya. Jika nilai hambatan delapan resistor tersebut membentuk pola barisan geometri, berapakah nilai hambatan resistor ke-5?


Jawab:

1250 ohm

Penjelasan :

Untuk mencari nilai hambatan resistor ke-5, kita perlu mengetahui rasio r dari pola barisan geometri tersebut terlebih dahulu. Kita dapat mencari rasio ini dengan membagi nilai hambatan resistor kedua dengan nilai hambatan resistor pertama:

r = hambatan resistor kedua / hambatan resistor pertama

r = 10 ohm / 2 ohm

r = 5

Dengan mengetahui rasio r, kita dapat mencari nilai hambatan resistor ke-5 dengan rumus umum untuk barisan geometri:

hn = h1 x  [tex]r^{n-1}[/tex]

Di mana:

hn = nilai hambatan resistor ke-n

h1 = nilai hambatan resistor pertama

n = urutan resistor yang dicari

Kita ingin mencari nilai hambatan resistor ke-5, sehingga:

hn = h1 x  [tex]r^{n-1}[/tex]

h5 = 2 ohm x  [tex]5^{5-1}[/tex]  

h5 = 2 ohm x  [tex]5^{4}[/tex]

h5 = 2 ohm x 625

h5 = 1250 ohm

Jadi, nilai hambatan resistor ke-5 adalah 1250 ohm.

24. resistor yang dipasang secara paralel maka nilai hambatannya makin ​


Jawaban:

Jika resistor-resistor dipasang secara paralel, nilai hambatan totalnya akan lebih kecil daripada hambatan terbesar di antara resistor-resistor tersebut. Dalam rangkaian paralel, arus listrik akan terbagi di antara resistor-resistor tersebut.

Dalam rangkaian paralel, hambatan total (Rt) dapat dihitung menggunakan rumus:

1/Rt = 1/R1 + 1/R2 + 1/R3 + ...

Di mana R1, R2, R3, dan seterusnya adalah nilai hambatan masing-masing resistor yang dipasang secara paralel.

Jadi, semakin banyak resistor yang dipasang secara paralel, semakin banyak jalur yang tersedia untuk arus listrik mengalir, sehingga hambatan total akan lebih kecil dari hambatan terbesar di antara resistor-resistor tersebut. Dengan kata lain, nilai hambatan total akan menurun ketika resistor-resistor dipasang secara paralel.


25. Jika sebanyak n buah resistor yang masing-masing memiliki nilai hambatan r dirangkai paralel, berapakah nilai hambatan pengganti bagi rangkaian resistor tersebut?​


Nilai hambatan pengganti bagi rangkaian tersebut adalah r/n. Simak penjelasan berikut.

Pembahasan

Hukum Ohm menyatakan bahwa jika arus listrik melalui suatu penghantar pada suhu tetap, maka kuat arusnya berbanding lurus dengan beda potensial pada kedua ujung penghantar tersebut. Secara matematis, pernyataan tersebut dirumuskan sebagai berikut.

R = V / I

dengan

V = tegangan (Volt)

I = kuat arus listrik (Ampere)

R = hambatan (Ω atau ohm)

A) Susunan resistor seri pada rangkaian

Hambatan total dari resistor yang dirangkai secara seri dapat ditentukan dengan rumus sebagai berikut.

Rs = R₁ + R₂ + R₃ + ...

Nilai arus dan tegangan pada susunan resistor seri

I = I₁ = I₂ = I₃ = .....

V = V₁ + V₂ + V₃ + ...

B) Susunan resistor paralel pada rangkaian

Hambatan total dari resistor yang dirangkai secara paralel dapat ditentukan dengan rumus sebagai berikut.

1/Rp = 1/R₁ + 1/R₂ + 1/R₃ + ...

Nilai arus dan tegangan pada susunan resistor paralel

I = I₁ + I₂ + I₃ + ...

V = V₁ = V₂ = V₃ = ...

Diketahui

Jumlah resistor = n

Nilai hambatan = r

Ditanya

Nilai hambatan pengganti jika disusun paralel, Rp

Penyelesaian

Rangkaian paralel memiliki hambatan pengganti

1/Rp = 1/R₁ + 1/R₂ + 1/R₃ + ...

1/Rp = 1/r + 1/r + 1/r + ...

Jika jumlah resistor = n, maka

1/Rp = n/r

Rp = r/n

Kesimpulan

Jadi, nilai hambatan pengganti bagi rangkaian tersebut adalah r/n.

33. sebuah resistor memiliki gelang berwarna merah coklat dan merah. tentukan nilai hambatan resistor


Jawaban:

Nilai hambatannya adalah 220Ω

Penjelasan:

Warna pada gelang resistor digunakan sebagai kode untuk menentukan nilai hambatannya. Dalam sistem kode warna 4 band, masing-masing warna menunjukkan sebuah digit angka dan warna ketiga menunjukkan toleransi, atau seberapa banyak hambatannya bisa berbeda dari nilai nominal.

Jika resistor memiliki gelang warna merah-coklat-merah, maka ini menunjukkan bahwa nilai hambatannya adalah 220Ω. Gelang pertama merah menunjukkan digit pertama, 2. Gelang kedua coklat menunjukkan digit kedua, 2. Gelang ketiga merah menunjukkan digit ketiga, 0. Jadi, secara keseluruhan, resistor tersebut memiliki nilai hambatan 220Ω.

Toleransi resistor ini harus ditentukan melalui gelang warna keempat, jika ada. Jika tidak ada gelang warna keempat, maka toleransinya biasanya ±20%.

diketahui:
R1 = 60 Ω
Rp = 20 Ω

ditanya:
R2?

jawab:
1/Rp = 1/R1 + 1/R2
1/R2 = 1/Rp - 1/R1
1/R2 = 1/20 - 1/60
1/R2 = 3/60 - 1/60
1/R2 = 2/60
R2 = 60/2
R2 = 30 Ω

jadi resistor yang lain adalah 30 Ω
