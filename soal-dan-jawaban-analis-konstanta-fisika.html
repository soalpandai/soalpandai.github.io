satuan gaya adalah kg . m / s^2
maka dimensinya menjadi M . L . T^-2
catatan : ^ dibaca pangkat ya

satuan arak adalah m
maka dimensinya L

untuk muatan satuannya adalah c
muatan adalah arus dikali waktu I . T dng satuan Ampere sekon (=coulomb)
maka dimensinya I . T

sekarang kalikan, maka didapat
k = (M . L . T^-2) . (L)^2 . (I . T)^-2
k = M . L^3 . I^-2 . T^-4

27. buatlah 3 soal beserta jawaban mengenai perkalian matriks dengan konstanta ​


Contoh Soal Perkalian Matriks

1. Hitunglah

( 1 2 ) x ( 3 2 )

( 1 3 ) x ( 4 1 ) = ?

Penyelesaian ;

( 1 2 ) x ( 3 2 )

( 1 3 ) x ( 4 1 )

= ( 1 x 3 + 2 x 4 1 x 2 + 2 x 1 )

( 1 x 3 + 3 x 4 1 x 2 + 3 x 1 )

= ( 11 14 )

( 15 5 )

2. Berapakah nilai x+y yang memenuhi

( x 1 ) x ( 3 2 ) = ( 7 4 )

( -1 y ) x ( 1 0 ) = ( -3 -2)

Penyelesaian:

( x 1 ) x ( 3 2 ) = ( 3x + 1 2x)

( -1 y ) x ( 1 0 ) = ( -3 + y -2)

( 3x + 1 2x) = ( 7 4 )

( -3 + y -2 ) = ( -3 -2 )

Sesuaikan persamaan terhadap posisi elemen, didapat

3x + 1 = 7 → x = 2 dan -3 + y = -3 → y = 0

Jadi, x + y = 2 + 0 = 2

3. Berapakah hasil dari

( 1 -7 ) x ( 5 -3 8 )

( 1 -7 ) x ( 5 -3 8 )

( 5 9 ) x ( 0 2 -1 )

=

( 1 x 5 + (-7) x 0 1x - 3 + (-7) x 2 1 x 8 + (-7) x-1 )

( 5 x 5 + 9 x 0 5x - 3 + 9 x 2 5 x 8 + 9 x -1 )

= ( 5 -17 15 )

30. apakah di UGM tersedia jurusan analis kesehatan?? mohon dijawab ya


Profesi Analis Kesehatan kuliah pada jurusan bernama Teknologi Laboratorium Medik. Sependek penelusuran di website resmi bagi yang hendak mendaftar di Universitas Gadjah Mada yakni um.ugm.ac.id, pada daftar jurusan yang tersedia TIDAK DIJUMPAI adanya jurusan Teknologi Laboratorium Medik yang menjadi jurusan bagi mereka yang berprofesi sebagai Analis Kesehatan.

» Pembahasan

Analis Kesehatan ini terbilang sebagai profesi yang ‘baru’ sehingga oleh awam masih terdengar asing. Istilah lain dari Analis Kesehatan ini adalah Pranata Laboratorium. Meski disebutkan bahwa jurusan yang tepat bagi profesi ini adalah Teknik Laboratorium Medik namun sumber lain menyatakan bahwa jurusan seperti Kesehatan masyarakat, Farmasi dan kimia juga bisa bekerja di bidang tersebut namun mungkin dengan syarat harus memenuhi kriteria tertentu.

Disebutkan pula bahwa terdapat jurusan bernama Analis Kesehatan namun keberadaannya sangat langka di Indonesia. Jika dihitung, jumlah institusi pendidikan yang menyediakan jurusan ini tidak lebih dari 20 dan terbatas hanya pada jenjang Diploma III saja contohnya di Politeknik Kesehatan Manado, Sulawesi Utara.

Perlu diketahui bahwa profesi Analis Kesehatan ini berbeda dengan bidan, perawat atau pun dokter. Mereka yang bekerja sebagai Analis Kesehatan ini hanya berkecimpung di dalam laboratorium dengan tugas utama yakni memeriksa, mengukur, menetapkan, dan menguji bahan-bahan yang asalnya dari manusia ataupun bukan berasal dari manusia dengan tujuan menetukan jenis suatu penyakit, penyebab munculnya penyakit tersebut, keadaan kesehatan perorangan atau pun masyarakat dan juga faktor-faktor yang bisa mempengaruhi kondisi kesehatan perseorangan/masyarakat.
