33. Diketahui vector a = 4i – 2j + 2k dan vector b = 2i – 6j + 4k. Proyeksi vector orthogonal vector a pada vector b adalah .....


vektor a = 4i – 2j + 2k

vektor b = 2i – 6j + 4k

a . b = (4).(2) + (-2).(-6) + (2).(4) = 8 + 12 + 8 = 28

|b|² = 2² + [-6]² + 4² = 4 + 36 + 16 = 56

Proyeksi vector orthogonal vector a pada vector b :

= ( [a . b] / |b|² ) . vektor b

= (28 / 56) . vektor b

= ½ . vektor b

= ½ . (2i – 6j + 4k)

= i - 3j + 2k


34. vector m memilik panjang 90 cm dengan arah 37 o terhadap sumbu x sedangkan vector n memilki panjang 50 cm dengan arah 53 o terhadap sumbu x .hitung besar dan arah vector resultan nya # soal fisika kelas 10# no ngasal >< report ​


R₁ = 90 cm

α₁ = 37° ➡ sin α₁ = ⅗ ➡ cos α₁ = ⅘

α₂ = 53° ➡ sin α₂ = ⅘ ➡ cos α₂ = ⅗

R₂ = 50 cm

pada sumbu X

Rx = R₁ cos α₁ + R₂ cos α₂

Rx = 90 × ⅘ + 90 × ⅗

Rx = 126 cm

pada sumbu Y

Ry = R₁ sin α₁ + R₂ sin α₂

Ry = 50 × ⅗ + 50 × ⅘

Ry = 70 cm

R = ___ ?

resultan vektor

R = √(Rx² + Ry²)

R = √(126² + 70²)

R = 14√106 cm ✔

arah vektor = __ ?

tan θ = Ry / Rx

tan θ = 70 / 126

θ ≈ 29,1° terhadap sumbu X ✔

Kelas : 10

Mapel : Fisika

Materi : Vektor

Kode Kategorisasi : 10.6.2

Kata Kunci : vektor, resultan, arah vektor
