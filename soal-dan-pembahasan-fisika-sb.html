35. tentukan fungsi kuadrat yang grafiknya memotong sb x di (-2,0) dan sb y di(0,8)​


Jawaban:

Untuk menentukan fungsi kuadrat yang grafiknya memotong sumbu x di (-2,0) dan sumbu y di (0,8), kita dapat menggunakan bentuk umum fungsi kuadrat y = ax^2 + bx + c.

Kita dapat menggunakan titik (0,8) untuk menentukan nilai c. Ketika x = 0, y = 8, sehingga kita memiliki persamaan:

8 = a(0)^2 + b(0) + c

8 = c

Jadi, nilai c adalah 8.

Selanjutnya, kita dapat menggunakan titik (-2,0) untuk menentukan hubungan antara a dan b. Ketika x = -2, y = 0, sehingga kita memiliki persamaan:

0 = a(-2)^2 + b(-2) + 8

0 = 4a - 2b + 8

-4a + 2b = -8

-2a + b = -4

Karena kita memiliki dua persamaan dengan dua variabel (a dan b), kita dapat menyelesaikannya dengan metode eliminasi atau substitusi. Dalam hal ini, kita akan menggunakan metode substitusi.

Dari persamaan sebelumnya, kita dapat mengisolasi b:

b = 2a - 4

Kemudian, kita substitusikan nilai b ini ke persamaan sebelumnya:

-2a + (2a - 4) = -4

-2a + 2a - 4 = -4

-4 = -4

Karena persamaan tersebut benar, kita dapat menyimpulkan bahwa fungsi kuadrat yang memenuhi syarat adalah:

y = ax^2 + (2a - 4)x + 8

Anda dapat memilih nilai a yang sesuai untuk mendapatkan fungsi kuadrat yang spesifik.

Jawab:

0 = 3x² - x - 10

Kita dapat menyelesaikan persamaan kuadrat ini dengan menggunakan rumus abc:

Penjelasan dengan langkah-langkah:

Untuk melakukan refleksi terhadap sumbu x, kita perlu mengubah tanda koordinat y dari titik-titik tersebut. Jadi, jika kita memiliki titik A (4, 2) dan titik B (-6, -4), setelah melakukan refleksi terhadap sumbu x, titik A akan menjadi (4, -2) dan titik B akan menjadi (-6, 4).

Selanjutnya, untuk melakukan refleksi terhadap sumbu y, kita perlu mengubah tanda koordinat x dari titik-titik tersebut. Jadi, setelah melakukan refleksi terhadap sumbu y, titik A akan menjadi (-4, -2) dan titik B akan menjadi (6, 4).

Dengan demikian, setelah melakukan refleksi terhadap sumbu x dan sumbu y, titik A akan menjadi (-4, -2) dan titik B akan menjadi (6, 4).
