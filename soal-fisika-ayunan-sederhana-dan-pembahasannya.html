1. Bagaimana jalan untuk mendapatkan hasil frekuensi ayunan dan priode ayunan soal: jika ayunan sederhana bergetar sebanyak 60 kali dalam waktu 15 sekon.Tentukan


a. Frekuensi getaran adalah [tex]\boxed{\bold{4 \: Hz}}[/tex]

b. Periode getaran adalah [tex]\boxed{\bold{0,25 \: s}}[/tex]

Pembahasan

Hi! soal tersebut adalah membahas mengenai getaran dan gelombang. Singkatnya, getaran adalah pergerakan berbolak balik di sekitar keseimbangan. Biasanya getaran di ilustrasikan dengan bandul atau gerakan pegas.

Gerakan bandul :

Jika bandul A-B = [tex]\bold{\frac{1}{4}}[/tex] getaran. Jika bandul A-B-C = [tex]\bold{\frac{1}{2}}[/tex] getaran. Jika bandul A-B-C-B = [tex]\bold{\frac{3}{4}}[/tex] getaran. Jika bandul A-B-C-B-A = [tex]\bold{1}[/tex] getaran.

Rumus menentukan frekuensi :

[tex]\boxed{\bold{f = \frac{n}{t}}}[/tex]

Dimana :

[tex]\bold{f}[/tex] = frekuensi.

[tex]\bold{n}[/tex] = jumlah getaran dalam waktu t sekon.

Berarti :

[tex]\boxed{\bold{f = \frac{1}{T}}}[/tex]

Rumus menentukan periode :

[tex]\boxed{\bold{T = \frac{t}{n}}}[/tex]

Dimana :

[tex]\bold{T}[/tex] = periode.

[tex]\bold{n}[/tex] = banyak getaran.

[tex]\bold{t}[/tex] = waktu.

Berarti :

[tex]\boxed{\bold{T= \frac{1}{f}}}[/tex]

Langkah Penyelesaian

Diketahui :

Banyak getaran = 60 kali Waktu = 15 sekon.

Ditanya :

a. Frekuensi getaran? b. Periode getaran?

Dijawab :

Mencari frekuensi :

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{60}{15}}[/tex]

[tex]\bold{f = 4 \: Hz}[/tex]

Mencari periode :

[tex]\bold{T = \frac{t}{n}}[/tex]

[tex]\bold{T = \frac{15}{60}}[/tex]

[tex]\bold{T = 0,25 \: s}[/tex]

Kesimpulan : Jadi frekuensi getaran adalah [tex]\boxed{\bold{4 \: Hz}}[/tex] sedangkan periode nya adalah [tex]\boxed{\bold{0,25 \: s}}[/tex]

24. periode ayunan sederhana dipengaruhi oleh


Periode ayunan bandul sederhana atau pendulum hanya bergantung kepada panjang tali dan percepatan gravitasi.

Pembahasan

Percobaan menggunakan ayunan bandul pernah dilakukan oleh Galileo di sebuah Katedral di Pisa, Italia. Setelah ditarik, bandul dilepas dan mengayun sehingga terjadi gerakan bolak-balik secara periodik atau timbul osilasi.

Dari hasil eksperimen yang berlangsung pada tahun 1602 tersebut, disimpulkan bahwa faktor-faktor di bawah ini tidak berpengaruh terhadap periode ayunan bandul, yakni:

bentuk dan ukuran ayunan;massa bandul di ujung ayunan;sudut ayunan bandul.

Periode ayunan bandul hanya bergantung kepada panjang tali dan percepatan gravitasi.

[tex]\boxed{~ T = 2\pi \sqrt{ \frac{L}{g} }~}[/tex]

Keterangan

T = periode (sekon)L = panjang tali (meter)g = percepatan gravitasi (m/s²)

Dari rumus di atas, tampak bahwa periode ayunan sederhana:

berbanding lurus dengan akar kuadrat panjang tali, dan berbanding terbalik dengan akar kuadrat percepatan gravitasi bumi.

Jadi, semakin panjang tali, semakin besar pula periodenya.

Lokasi percobaan yang berbeda di berbagai permukaan bumi dapat menimbulkan sedikit perbedaan periode ayunan bandul dengan panjang tali yang sama, namun hal ini tidak begitu signifikan.

___________________

Catatan:

Rumus di atas dapat juga digunakan untuk menghitung percepatan gravitasi. Setelah diolah, rumus untuk menentukan percepatan gravitasi dari percobaan ayunan bandul adalah sebagai berikut:

[tex]\boxed{ \ g = \Big( \frac{4 \pi^2}{T^2} \Big ) \ }[/tex]

37. Jika ayunan sederhana bergetar sebanyak 50 kali dalam waktu 10 sekon. Maka besar frekuensi ayunan dan periode ayunannya adalah...


Jika ayunan sederhana bergetar sebanyak 50 kali dalam waktu 10 sekon , maka besar frekuensi ayunan dan periode ayunannya adalah 5 Hz dan 0,2 sekon

PEMBAHASAN

Gelombang sinusoidal adalah gelombang yang membentuk fungsi sinus. Beberapa rumus dasar gelombang yang perlu di ingat adalah sebagai berikut ini :

f = 1 / T

v = λ . f

Δx / λ = ΔΘ / 2π

Δt / T = ΔΘ / 2π

dimana :

f = frekuensi gelombang ( Hz )

T = periode gelombang ( s )

v = cepat rambat gelombang ( m/s )

λ = panjang gelombang ( m )

Δx = jarak pisah antar dua titik pada gelombang ( m )

Δt = selang waktu ( s )

ΔΘ = beda fase ( radian )

Bentuk persaman gelombang adalah :

y = A sin ( ωt - kx )

dimana :

y = simpangan gelombang ( m )

A = amplitudo gelombang ( m )

ω = frekuensi sudut ( rad/s )

t = waktu ( s )

k = bilangan gelombang ( 1/m ) = 2π/λ

x = jarak titik dari sumber ( m )

Okay mari kita gunakan rumus ini untuk menyelesaikan soalnya.

Diketahui :

Jumlah getaran = n = 50 kali

Waktu getaran = t = 10 sekon

Ditanyakan :

f = ?

T = ?

Penyelesaian :

f = n / t

f = 50 / 10

f = 5 Hz

T = t / n

T = 10 / 50

T = 0,2 sekon
