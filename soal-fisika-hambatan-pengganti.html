a. R = 2 + 2 + (6 // 6) = 2 + 2 + 3 = 7 ohm
b. R = 2 + ((4+4) // 8) = 2 + (8 // 8) = 2 + 4 = 6 ohm
c. R = (9 // 9 // 9) + 2 = 3 + 2 = 5 ohm
d. R = (5 // 10) + (2 // 2) = 3,33 + 1 = 4, 33 ohm

2. yang jago fisika jawab ya!!! pada rangkaian hambatan di bawah ini R1=R2=R3=10 OHM.Nilai hambatan pengganti paling kecil terdapat pada rangkaian .... (soalnya pilihan ganda cari hambatan paling kecil jawab pakai cara )


jelas , rangkaian pararel nya yg hambatan penggantinya kecil jadi A Salah 

B = 10 + 10/3 
   = 10 + 3,33
   = 13,33 

C = 20 + 5 
   = 25

D = 1/10 + 1/10 + 1/10 +1/10 
   = 4/10 
    = 10/4 
   =2,5 


jadi yang paling kecil itu yang Da. R = R1+R2+R3+R4 = 10Ω+10Ω+10Ω+10Ω = 40Ω

b. R = R1+((R2xR3xR4):(R2+R3+R4)) = 10Ω+((10Ωx10ΩX10Ω):(10Ω+10Ω+10Ω)) = 10Ω+(1000Ω:30Ω) = 10Ω+33,33Ω = 43,33Ω

c. R = R1+((R2xR3):(R2+R3))+R4 = 10Ω+((10Ωx10Ω):(10Ω+10Ω))+10Ω = 10Ω+(100Ω:20Ω)+10Ω=10Ω+5Ω+10Ω=25Ω

d. 1/R = 1/R1+1/R2+1/R3+1/R4 
    1/R = 1/10Ω+1/10Ω+1/10Ω+1/10Ω
    1/R = 4/10Ω
     R   = 10/4Ω
     R   = 2,5Ω

Di antara semua hasil, d adalah yang paling kecil. 

Jadi, jawabannya adalah d. 

Jika salah, mohon maaf yang sebesar-besarnya. 



3. Soal Fisika. Perhatikan rangkaian berikut ini : R₁ = 4 R₂ = 2 R₃ = 6 R₄ = 6 T - V = 4 volt. Hitunglah penghambatan pengganti dan kuat arus yang mengalir! Mohon pake cara dengan benar.


Rp = R3 + R4 = 1/6 +1/6 = 2/6 = 6/2 = 3 Rs=3+4+2 = 9 ohm I=V/R =4/9 = 0,4 ADiket : R1 = 4Ω
           R2 = 2Ω
           R3 = R4 = 6Ω
           V = 4 Volt
Ditanya : a. R pengganti = ... ?
               b. I = .... ?
Jawab : a. R pengganti :
// selesaikan R paralel (R3 dan R4)
1/Rp = 1/R3 + 1/R4
1/Rp = 1/6 + 1/6
1/Rp = 2/6
Rp = 6/2
Rp = 3Ω
// Cari R pengganti :
Rs = R1 + R2 + Rp
Rs = 4 + 2 + 3 = 9Ω

b. I = V/R
    I = 4/9
    I = 0,44 A
