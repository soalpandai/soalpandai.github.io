8. Terdapat 176 siswa SMA Unggul yang diterima di perguruan tinggi negeri (PTN) melalui jalur SNMPTN, SBMPTN dan seleksi mandiri. Jumlah siswa yang diterima melalui jalur seleksi mandiri 52 siswa lebih banyak dari jumlah siswa yang diterima jalur SBMPTN. jika jumlah siswa yang diterima melalui jalur SNMPTN dua kali dari jumlah siswa yang diterima melalui jalur SBMPTN, maka jumlah siswa yang diterima melalui jalur seleksi mandiri adalah.......


mandiri = 52 + SBMPTN
SNMPTN = 2 x SBMPTN

mandiri + SBMPTN + SNMPTN = 176
52 + SBMPTN + SBMPTN + 2 SBMPTN = 176
4 SBMPTN + 52  = 176
4 SBMPTN = 176 - 52
                   = 124
SBMPTN = 124/4 = 31
mandiri = 52 + SBMPTN
             = 52 + 31
             = 83

Tugas Mandiri

Hitung Luas permukaan dan Volume Prisma segitiga berikut ini :

1. Luas permukaan = 220 + 120[tex]\sqrt{5}[/tex] cm²

Volume = 500 cm³

2. Luas permukaan = 396cm²

Volume = 432cm³

3. Luas permukaan = 1.152cm²

Volume = 1.920cm³

4. Luas permukaan = 888cm²

Volume = 1.056cm³

5. Luas permukaan = 360cm²

Volume = 300cm³

Rumus Mencari Luas permukaan dan Volume prisma segitiga :

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

Pembahasan :

1. Diketahui :

alas segitiga = 10cm

tinggi segitiga = 10cm

tinggi prisma = 12cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya, karena ini segitiga sama kaki maka sisi yg lainnya sama besar

sisi = [tex]\sqrt{(\frac{1}{2} \times 10)^2 + 10^2}[/tex]

sisi = [tex]\sqrt{25 + 100}[/tex]

sisi = [tex]\sqrt{125}[/tex]

sisi = 5[tex]\sqrt{5}[/tex] cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 10 x 10) + (10 x 12) + 2(5[tex]\sqrt{5}[/tex] x 12)

= 100 + 120 + 120[tex]\sqrt{5}[/tex]

= 220 + 120[tex]\sqrt{5}[/tex] cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 10 x 10 x 12

= 500 cm³


2. Diketahui :

alas segitiga = 9cm

tinggi segitiga = 12cm

tinggi prisma = 8cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya (segitiga siku-siku)

sisi miring = [tex]\sqrt{9^2 + 12^2}[/tex]

sisi miring = [tex]\sqrt{81 + 144}[/tex]

sisi miring = [tex]\sqrt{225}[/tex]

sisi miring = 15cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 9 x 12) + (9 x 8) + (12 x 8) + (15 x 8)

= 108 + 72 + 96 + 120

= 396cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 9 x 12 x 8

= 432cm³


3. Diketahui :

alas segitiga = 12cm

tinggi segitiga = 16cm

tinggi prisma = 20cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya (kita anggap segitiga siku-siku)

sisi miring = [tex]\sqrt{16^2 + 12^2}[/tex]

sisi miring = [tex]\sqrt{256 + 144}[/tex]

sisi miring = [tex]\sqrt{400}[/tex]

sisi miring = 20cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 12 x 16) + (12 x 20) + (16 x 20) + (20 x 20)

= 192 + 240 + 320 + 400

= 1.152cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 12 x 16 x 20

= 1.920cm³


4. Diketahui :

alas segitiga = 16cm

tinggi segitiga = 6cm

tinggi prisma = 22cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya, karena ini segitiga sama kaki maka sisi yg lainnya sama besar

sisi = [tex]\sqrt{(\frac{1}{2} \times 16)^2 + 6^2}[/tex]

sisi = [tex]\sqrt{64 + 36}[/tex]

sisi = [tex]\sqrt{100}[/tex]

sisi = 10cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 16 x 6) + (16 x 22) + 2(10 x 22)

= 96 + 352 + 440

= 888 cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 16 x 6 x 22

= 1.056 cm³


5. Diketahui :

alas segitiga = 5cm

tinggi segitiga = 12cm

26. bisa jelaskan apa itu jalur umum dan jalur mandiri yg biasa ada di pendaftaran❓​


Jalur untuk masuk perkuliahan ada dua yaitu Jalur Umum dan Jalur Mandiri. Jalur Umum terdiri dari 2 Jalur lagi yaitu Jalur SBMPTN (Seleksi Bersama Masuk Perguruan Tinggi Negeri) dan SNMPTN (Seleksi Nasional Masuk Perguruan Tinggi Negeri). Nah SBMPTN itu cara masuknya menggunakan nilai UTBK (Ujian Tulis Berbasis Komputer) kemudian diranking se indonesia. Kurang lebih sama prinsipnya dengan Ujian Nasional. Sedangkan SNMPTN penyeleksiannya berdasarkan Nilai Raport. Jalur Mandiri Kurang lebih mekanismenya seperti UTBK namun diadakan secara mandiri dari PTN pilihan dengan Biaya Tambahan dari Pihak Siswa.

Jalur mandiri, merupakan seleksi berikutnya saat hendak masuk PTN. Bila belum bisa melalui SNMPTN dan SBMPTN

Di dalam tes ini, calon mahasiswa akan diuji kemampuan verbal, numerik, dan logika mereka.Tes Wawasan Kebangsaan (TWK): Tes ini meliputi materi Pancasila, Sejarah Indonesia, Kewarganegaraan, dan Bahasa Indonesia. Di dalam tes ini, calon mahasiswa akan diuji pengetahuan dan pemahaman mereka tentang wawasan kebangsaan dan kebhinekaan Indonesia.
