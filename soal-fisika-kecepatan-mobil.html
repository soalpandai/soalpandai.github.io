diketahui:
vo = 0 m/s (mula diam)
a = α
-a = β
waktu total = t

ditanya:
vmax?

jawab:
Selang waktu ketika dipercepat
vt = vo + a t1
vt = 0 + α t1
t1 = vt / α

Selang waktu ketika diperlambat (vt = 0 & vo = vt)
vt = vo - a t2
0 = vt - β t2
t2 = vt / β

Kecepatan maksimum
t1 + t2 = vt/α + vt/β
t = vt/α + vt/β
t = vtβ/αβ + vtα/αβ
t = vt (α + β) / αβ
vt = αβt / (α + β)

jadi kecepatan maksimumnya adalah αβt / (α + β)

m = 900 kg
P = 8 pk = 8 x 750 = 6000 watt
Vo = 36 km/jam = 10 m/s

P . t = 1/2 . m . (Vt^2 - Vo^2)

v₂² = v₁² + 2aΔx
v₂² - v₁² = 2aΔx
(v₂² - v₁²) / 2 = aΔx ... (**)

FΔx = m[(v₂² - v₁²) / 2]
