Listrik muncul akibat adanya beberapa tipe fisika:

1. Muatan listrik: sifat beberapa partikel subatomik yang menentukan interaksi elektromagnetik. Substansi yang bermuatan listrik menghasilkan dan dipengaruhi oleh medan elektromagnetik

2 . Medan listrik (lihat elektrostatis): tipe medan elektromagnetik sederhana yang dihasilkan oleh muatan listrik ketika diam (maka tidak ada arus listrik). Medan listrik menghasilkan gaya ke muatan lainnya

3. Potensial listrik: kapasitas medan listrik untuk melakukan kerja pada sebuah muatan listrik, biasanya diukur dalam volt

4. Arus listrik: perpindahan atau aliran partikel bermuatan listrik, biasanya diukur dalam ampere

5. Elektromagnet: Muatan berpindah menghasilkan medan magnet. Arus listrik menghasilkan medan magnet dan perubahan medan magnet menghasilkan arus listrik.

Sebuah Kereta Api Listrik berangkat dari stasiun A hingga stasiun B. Jarak antara stasiun A dan stasiun B adalah 799 meter. Berikut penyelesaiannya
PEMBAHASAN

Diketahui

a₁= 1 m/s²

t₁ = 6 sekon

a₂ = 2 m/s²

Vt₂ = 20 m/s

t₄ = 5 sekon


Ditanya

stotal = ?


PENYELESAIAN

Peristiwa dibagi menjadi 4 bagian (perhatikan gambar).


Pada peristiwa 1, mulanya, kereta api diam sebelum berangkat. Maka Vo₁ = 0


• cari jarak tempuh peristiwa 1

s₁ = Vo₁ • t + ½ • a • t₁²

s₁ = 0 • 6 + ½ • 1 • 6²

s₁ = 0 + 18

s₁ = 18 meter


• cari kecepatam akhir bagian 1

Vt₁ = Vo₁ + a • t₁

Vt₁ = 0 + 1 • 6

Vt₁ = 6 m/s


Peristiwa 2, kereta api bertambah cepat. Disini, kecepatan akhir peristiwa 1 akan menjadi kecepatan awal peristiwa 2.


Vt₁ = Vo₂

Vt₂ = 20 m/s


• cari percepatan peristiwa 2

Vt₂² = Vo₂² + 2 • a • s₂

20² = 6² + 2 • 2 • s₂

400 = 36 + 4s₂

4s₂ = 364

s₂ = 91 meter


• cari waktu tempuh peristiwa 2

Vt₂ = Vo₂ + a • t₂

20 = 6 + 2 • t₂

14 = 2t₂

t₂ = 7 sekon


Peristiwa 3, kereta api bergerak lurus beraturan. Dimana kecepatan akhir peristiwa 2 akan menjadi kecepatan konstan pada peristiwa 3.


Vt₂ = V₃


• cari waktu tempuh peristiwa 3

t = t₁ + t₂ + t₃ + t₄

50 = 6 + 7 + t₃ + 5

50 = 18 + t₃

t₃ = 32 sekon


kereta api bergerak konstan selama 32 sekon.


• cari jarak tempuh peristiwa 3

s₃ = v₃ • t₃

s₃ = 20 • 32

s₃ = 640 meter


Peristiwa 4, kereta api melambat. Dimana kecepatan peristiwa 3 menjadi kecepatan awal di peristiwa 4.


V₃ = Vo₄

t = 5 sekon

Vt₄ = 0 (kereta api berhenti)


• cari perlambatan peristiwa 4

Vt₄ = Vo₄ + a • t₄

0 = 20 + a • 5

-5a = 20

a = -4 m/s²


• cari jarak tempuh peristiwa 4

Vt₄² = Vo₄² + 2 • a • s₄

0² = 20² + 2 • -4 • s₄

0 = 400 - 8s₄

8s₄ = 400

s₄ = 50 meter


ㅡㅡㅡ

• cari jarak total

stotal = s₁ + s₂ + s₃ + s₄

stotal = 18 + 91 + 640 + 50

stotal = 799 meter

PEMBAHASAN MATERI GLB DAN GLBB

Gerak Lurus dapat dibagi menjadi 2 bagian, yaitu Gerak Lurus Beraturan (GLB) dan Gerak Lurus Berubah Beraturan (GLBB).


[tex] \boxed {GLB} [/tex]

         GLB merupakan gerak lurus beraturan. Dimana kecepatan tidak mengalami percepatan ataupun perlambatan. Kecepatan akan terus konstan.


[tex] \boxed {v = \frac {s}{t}} [/tex]


v = kecepatan (m/s)

s = jarak yang ditempuh (m)

t = waktu yg diperlukan (s)


[tex] \boxed {GLBB} [/tex]

         GLBB merupakan gerak lurus berubah beraturan. Disini terdapat unsur percepatan yang membedakan antara GLB dan GLBB. GLBB memiliki percepatan, sehingga benda yg awalnya berkecepatan awal Vo akan berubah menjadi Vt baik itu bertambah cepat atau pun bertambah lambat.


[tex] \boxed { Vt^{2} = Vo^{2} + 2as} [/tex]

[tex] \boxed { s = Vot + \frac{1}{2} at^{2}} [/tex]

[tex] \boxed { Vt = Vo + at } [/tex]


Vo = Kecepatan awal (m/s)

Vt = Kecepatan akhir (m/s)

a = percepatan (m/s²)

s = jarak yg ditempuh (m)

t = waktu yg diperlukan (s)

[tex] \sf \frac{1}{ R_{paralel}} = \frac{1}{ R_1 +{ R_2} } + \frac{1}{ R_3}[/tex]

10. mohon bantuan nya.. soal fisika


Soal Fisika

Cepat rambat gelombang tersebut adalah 8 m/s.

Pembahasan

Berdasarkan gambar tersebut, diketahui :

Dalam jarak 6 m, terjadi 1,5 gelombang. Panjang gelombangnya, λ = 6 / 1,5 = 4 m.

Selang waktu terjadinya 1 bukit dan 1 lembah atau disebut 1 gelombang, t = 0,5 s. Frekuensi gelombang, f = 1 / 0,5 = 2 Hz.

Ditanya :

Cepat rambat gelombang tersebut

Jawab :

Menghitung cepat rambat gelombang dengan persamaan

v = λ × f

v = 4 × 2

v = 8 m/s

Jadi cepat rambat gelombang tersebut adalah 8 m/s.

Soal nomor 10

HUKUM ARCHIMEDES

Diketahui:

Volume balok, V = 4.10⁻⁵ m³

Berat di udara, w(udara) = 1 N

Percepatan gravitasi, g = 10 m/s²

Massa jenis air, ρ = 1000 kg.m⁻³

Ditanya:

Berat di dalam air, w(air) = ...?

Penyelesaian:

w(udara) - w(air) = ρ.V.g

1 - w(air) = 1000.4.10⁻⁵.10

1 - w(air) = 0,4

w(air) = 1 - 0,4

w(air) = 0,6 N

Soal nomor 11

GERAK VERTIKAL

Diketahui:

Bola A dilempar vertikal, tinggi maksimum h.

Bola B dijatuhkan tanpa kecepatan awal dari ketinggian H

Bola A dan B jatuh bersamaan, tA = tB

Ditanya:

Tinggi bola B, H = ... h

Penyelesaian:

Pada bola A berlaku

h = vo²/2g

vo = √(2gh)

Lamanya bola A di udara

tA = 2.vo/g

tA = 2√(2gh)/g

tA = 2√(2h/g)

Pada bola B berlaku

H = 1/2.g.tB²

tB = √(2H/g)

Kedua bola jatuh ke tanah bersamaan, berlaku

tA = tB

2√(2h/g) = √(2H/g)

4.2h/g = 2H/g

4h = H

Jawaban: C

26. Soal fisika rangakaian listrik dengan amperemeter Dan voltmeter


Amperemeter
Contoh Soal :
Galvanometer memiliki hambatan 140 ohm untuk arus 1,2 mA agar menghasilkan simpangan dengan skala penuh. Berapa hambatan shunt yang harus dipakai  agar menghasilkan amperemeter yang memberikan simpangan skala penuh untuk arus 2A.
Jawaban :
Arus yang melalui Galvanometer IG=  0,0012 A
Arus total yang melalui amperemeter i = 2A 

iG RG  = I sh R sh

i= Ish   +  I G

ish = i – iG  = 2 – 0,0012 = 1,9988 ~ 2A

Nilai hambatan shunt

R sh = ( iG/I sh  )  RG  = (0,0012/2) . 140 = 0,084 ohm 


Voltmeter
Contoh soal:
Galvanometer memiliki hambatan  140 ohm untuk arus 1,2 mA sehingga memberikan simpangan skala penuh. Berapa besar hambatan yang dipasang secara seri untuk merancang sebuah voltmeter yang mampu memberikan simpangan skala penuh pada beda potensial 5 volt?
Jawab :
I (RS  +  RG)   = 5 volt
0,0012  (Rs + 140) = 5 volt
Rs + 140  = 4167 ohm
Rs = 4027 ohm
