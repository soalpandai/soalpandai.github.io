4. Soal menantang Fisika


Gerak translasi:
Tinjau benda m:
[tex]m g - T_1 = m a[/tex]
[tex]T_1 = m g - m a[/tex]

Gerak rotasi:
Tinjau bola M:
[tex]T_2 R = I \alpha[/tex]
[tex]T_2 R = \frac{2}{3} M R^2 \frac{a}{R}[/tex]
[tex]T_2 = \frac{2}{3} M a[/tex]

Tinjau katrol:
[tex](T_1 - T_2) r = I \alpha[/tex]
[tex](T_1 - T_2) r = I \frac{a}{r}[/tex]
[tex]T_1 - T_2 = I \frac{a}{r^2}[/tex]
[tex]m g - m a - \frac{2}{3} M a = I \frac{a}{r^2}[/tex]
[tex]m g = I \frac{a}{r^2} + m a + \frac{2}{3} M a[/tex]
[tex]m g = (\frac{I}{r^2} + m + \frac{2}{3} M) a[/tex]
[tex]a = \frac{m g}{(\frac{I}{r^2} + m + \frac{2}{3} M)}[/tex]
-----------------------------------------------------------------------------------
kelajuan benda m setelah menempuh jarak h:
[tex]v_1^2 = v_0^2 + 2 a s[/tex]
[tex]v_1^2 = 0 + 2 \frac{m g}{(\frac{I}{r^2} + m + \frac{2}{3} M)} h[/tex]
[tex]v_1 = \sqrt{2 \frac{m g}{(\frac{I}{r^2} + m + \frac{2}{3} M)} h}[/tex]

Penjelasan dengan langkah-langkah

Dikatehui:

1. Bejana air memiliki luas penampang 25 m². Jika bejana air dipanasi dengan api bersuhu 100° C dan suhu awal tungku 25° C,

2. Dua batang logam A dan B disambungkan dengan suhu ujung A = 150° C dan ujung B = 20° C. Koefisien konduktivitas logam A adalah kali koefisien konduktivitas logam B, luas permukaan kedua logam sama, dan pang jang A tiga kali panjang

3. Suatu gas dalam ruang tertutup menempati volume 2 liter pada tekanan tetap dan suhunya 27°C. Gas tersebut dipanaskan sehingga mencapai suhu 177° C.

4. Gas ideal berada dalam suatu ruang pada mulanay mempunyai volume V dan suhu T. Jika gas dipanaskan sehingga suhunya berubah menjadi 5/4 T dan tekanan berubah menjadi 2P maka

5. Sebuah kawat luas penampang 4 mm² dan panjangnya 10 m. Kawat di renggangkan dengan gaya 96 N. Jika modulus elastis kawat 25 x 1012 N/m²

Ditanya:

1. Tentukanlah:

a. Koefisien konveksi jika laju kalor 15 J/s

b. Kalor yang diperlukan untuk menaikkan suhu selama 5 menit

2. Tentukanlah suhu pada bidang persambungannya!

3. Tentukanlah volume gas tersebut!

4. Volume gas berubah menjadi?

5. Tentukanlah pertambahan pannjang kawat!

Jawab:

Langkah 1

No. 1

a. Koefisien konveksi:

h = q / ΔT

= 15 / (100 - 25)

= 0,2 J/s m²°C

b. Kalor yang diperlukan:

Q = h / t

= 0,2 / (5 x 60)

= 0,2 / 300

= 6,67 x 10⁻⁴ J

No. 2

Suhu pada bidang sambungannya adalah:

H A = H B

K A (TP - TQ) / l A = K B (TQ - TR) / l B

[tex]\frac{1}{2}[/tex] K B (150 - TQ) / 3 l B = K B (TQ - 20) / l B

TQ = 36,6 ° C

No. 3

Volume gas tersebut adalah:

V₁ / T₁ = V₂ / T₂

2 / (27 + 273) = V₂ / (177 + 273)

300 V₂ = 900

V₂ = 3 L

Langkah 2

No. 4

Persamaan: (P x V) /T

Maka, volume gas tersebut berubah menjadi:

(P₁ x V₁) / T₁ = (2 P₁ x V₂) / (5/4 T₁)

V₁  =2 V₂ / (5/4)

V₁ = 8/5 V₂

V₂ = 5/8 V₁

No. 5

γ = (F x l₀) / A x Δl)

Maka, pertambahan panjang kawat tersebut adalah:

Δl = (F x l₀) / (A x γ)

= (96 x 10) / (4 x 10⁻⁶ x 25 x 10¹²)

= 9,6 x 10⁶ m

14. Soal fisika dan caranya


f ob = 7,5 mm
s ob = 8 mm
f ok = 5 cm
s' ob = (s ob * f ob) / (s ob - f ob)
s' ob = (8 * 7,5) / (8 - 7,5) = 120 mm =12 cm

Untuk mata tidak berakomodasi
s' ok = - ∞
s ok = f ok = 5 cm
d = s' ob + s ok
d = 12 cm + 5 cm = 17 cm

Untuk mata berakomodasi maksimum
s' ok = -Sn        (Sn = jarak baca normal, biasanya 25 cm)
s' ok = -25 cm
s ok = (s' ok * f ok) / (s' ok - f ok)
s ok = (-25 * 5) / (-25 - 5) = 4,17
d = s' ob + s ok
d = 12 cm + 4,17 cm = 16,17 cm

30. Soal Fisika jawabannya harus fisika​


Jawaban:

C. 38.000 J

Penjelasan:

Diketahui :

massa tembaga (m) = 2 kg

perubahan suhu (ΔT) = 50° C

kalor jenis tembaga (c) = 380 J / kg°C

Ditanyakan : Kalor yang diperlukan (Q)

Jawab :

Rumus kalor : Q = m × c × ΔT

Q = m × c × ΔT

Q = 2 kg × 380 J/kg°C × 50°C

Q = 38.000 J

Jadi, besar kalor yang diperlukan adalah 38.000 J.
