5. Selesaikan soal dibawah ini! a. 192.168.0.0/26 Tentukan Subnet Mask, Host, IP Host awal, IP Host Akhir, IP Broadcast! b. 192.168.0.1/29 Tentukan Subnet Mask, Host, IP Host awal, IP Host Akhir, IP Broadcast!​di kerjakan di selembar kertas lah kk


a. 192.168.0.0/26

Subnet Mask = 255.255.255.192

Host = 62

IP Host Awal = 192.168.0.1

IP Host Akhir = 192.168.0.62

IP Broadcast = 192.168.0.63

b. 192.168.0.1/29

Subnet Mask = 255.255.255.248

Host = 6

IP Host Awal = 192.168.0.1

IP Host Akhir = 192.168.0.6

IP Broadcast = 192.168.0.7

Pembahasan

Subnetting dilakukan untuk mengelola jaringan dalam sebuah konfigurasi, contohnya di sebuah kantor, untuk membagi jaringan tersebut pada beberapa jalur seperti jalur di beberapa bagian kantor tersebut. Selain itu, tujuan lain dari subnetting dari sebuah jaringan adalah optimalisasi dan efisiensi kerja jaringan, karena jalur lalu lintas tidak terpusat di satu network besar, tapi terbagi ke beberapa ruas-ruas jaringan.

1) Apa yang kamu ketahui tentang jurusan instalasi listrik?

Jawab: Jurusan instalasi listrik adalah suatu keahlian yang mempelajari tentang bagaimana cara membangkitkan, menggunakan, mengubah, mengatur, mengalihkan dan mengumpulkan tenaga listrik.

2) Sebutkan tiga peralatan instalasi listrik beserta fungsinya yang kamu ketahui?

Jawab:

Sekring adalah peralatan listrik yang berfungsi untuk memutuskan aliran listrik apabila terjadi kelebihan daya listrik atau terjadi hubungan arus pendek.Sakelar adalah peralatan listrik yang berfungsi untuk menyalakan dan mematikan aliran listrik.Fiting adalah tempat yang digunakan untuk meletakkan sebuah lampu.

3) Apa alasan kamu memilih masuk SMK jurusan instalasi listrik?

Jawab: SMK jurusan instalasi listrik memiliki prospek kerja yang bagus seperti dapat bekerja sebagai ahli elektronika, ahli robotik, ahli penerbangan dan dapat bekerja di perusahaan pembangkit tenaga listrik dan lain - lain.

4)  Menurut kamu, apa bahan yang baik untuk pemasangan instalasi listrik di rumah?

Jawab: Bahan yang bersifat isolator untuk menghindari terjadinya korsleting listrik dan kebakaran akibat arus pendek listrik.

5) Menurut kamu, bagaimana perancangan instalasi listrik yang baik di dalam rumah?

Jawab: Peralatan instalasi listrik seperti sakelar, sekring, stop kontak, kabel listrik dan lain - lain harus berada pada posisi yang tepat atau strategis dan jauh dari jangkauan anak - anak untuk menghindari bahaya korsleting listrik.

21. 172.16.74.56 / 18soal:a. Jumlah Subnetb. Host Per Subnetc. Alamat Subnet markd. Blok subnete. Alamat Host & Broadcast valid1.Subnet2.Host 13.Host Akhir4.Broadcast​


Jawaban:

Dalam notasi IP Address yang diberikan (172.16.74.56 / 18), angka "18" menunjukkan panjang maska subnet dalam bit. Mari kita jawab setiap pertanyaan:

a. Jumlah Subnet:

Jumlah subnet dapat dihitung menggunakan rumus 2^n, di mana n adalah panjang maska subnet dalam bit. Dalam kasus ini, n=18, sehingga jumlah subnet adalah 2^18=262,144 subnet.

b. Host Per Subnet:

Jumlah host per subnet dapat dihitung menggunakan rumus 2^n−2, di mana n adalah panjang maska subnet dalam bit. Dalam kasus ini, n=18, sehingga host per subnet adalah 2^18−2=262,142 host.

c. Alamat Subnet Mark:

Alamat subnet mark adalah alamat IP pertama dalam subnet. Dalam notasi IP, bit host yang terakhir akan diatur ke 0. Jadi, alamat subnet mark adalah 172.16.64.0.

d. Blok Subnet:

Blok subnet adalah kisaran alamat IP yang dapat dialokasikan dalam subnet. Dalam kasus ini, dengan panjang maska 18 bit, blok subnet dimulai dari alamat subnet mark (172.16.64.0) hingga alamat broadcast terakhir dalam subnet. Alamat broadcast adalah alamat dengan semua bit host terakhir diatur ke 1. Jadi, blok subnet adalah 172.16.64.0 hingga 172.16.127.255.

e. Alamat Host & Broadcast Valid:

   Alamat Host Pertama: 172.16.64.1

   Alamat Host Terakhir: 172.16.127.254

   Alamat Broadcast: 172.16.127.255

Langkah Penyelesaian:

*Misalkan f(x) = y

[tex]y = \frac{x - 3}{2x + 1}\\\\ y(2x + 1) = x - 3\\\\2xy + y = x - 3\\\\2xy - x=-y-3\\\\x(2y - 1) = -y-3\\\\x = \frac{-y - 3}{2y-1} \\\\\boxed{x = f^{-1}(x) = \frac{- x - 3}{2x - 1} }[/tex]

*Substitusi nilai

[tex]f^{-1}(4) =  \frac{- x - 3}{2x - 1}\\\\f^{-1}(4) = \frac{- 4 - 3}{2(4) - 1}\\\\f^{-1}(4) = \frac{- 7}{7}\\\\f^{-1}(4) = - 1\\\\\therefore\boxed{ f^{-1}(4) = - 1}[/tex]
