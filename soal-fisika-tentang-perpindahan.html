F = (F1 + F2) - F3
   = (40N + 70N) - 50N = 110N - 50N = 60N
W = F x S
W = 60N x 3m = 180JDiketahui :
F1 = 40 N kiri
F2 = 70 N kiri
F3 = 50 N kanan
s = 3 m
Ditanyakan : Usaha ?

Dijawab :
W = ΣF x s
    = [(40+70) - 50] x 3
    = (110-50) x 3
    = 60 x 3
