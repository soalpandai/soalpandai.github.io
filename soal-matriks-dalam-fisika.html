4. Gas ideal berada dalam suatu ruang pada mulanay mempunyai volume V dan suhu T. Jika gas dipanaskan sehingga suhunya berubah menjadi 5/4 T dan tekanan berubah menjadi 2P maka

5. Sebuah kawat luas penampang 4 mm² dan panjangnya 10 m. Kawat di renggangkan dengan gaya 96 N. Jika modulus elastis kawat 25 x 1012 N/m²

Ditanya:

1. Tentukanlah:

a. Koefisien konveksi jika laju kalor 15 J/s

b. Kalor yang diperlukan untuk menaikkan suhu selama 5 menit

2. Tentukanlah suhu pada bidang persambungannya!

3. Tentukanlah volume gas tersebut!

4. Volume gas berubah menjadi?

5. Tentukanlah pertambahan pannjang kawat!

Jawab:

Langkah 1

No. 1

a. Koefisien konveksi:

h = q / ΔT

= 15 / (100 - 25)

= 0,2 J/s m²°C

b. Kalor yang diperlukan:

Q = h / t

= 0,2 / (5 x 60)

= 0,2 / 300

= 6,67 x 10⁻⁴ J

No. 2

Suhu pada bidang sambungannya adalah:

H A = H B

K A (TP - TQ) / l A = K B (TQ - TR) / l B

[tex]\frac{1}{2}[/tex] K B (150 - TQ) / 3 l B = K B (TQ - 20) / l B

TQ = 36,6 ° C

No. 3

Volume gas tersebut adalah:

V₁ / T₁ = V₂ / T₂

2 / (27 + 273) = V₂ / (177 + 273)

300 V₂ = 900

V₂ = 3 L

Langkah 2

No. 4

Persamaan: (P x V) /T

Maka, volume gas tersebut berubah menjadi:

(P₁ x V₁) / T₁ = (2 P₁ x V₂) / (5/4 T₁)

V₁  =2 V₂ / (5/4)

V₁ = 8/5 V₂

V₂ = 5/8 V₁

No. 5

γ = (F x l₀) / A x Δl)

Maka, pertambahan panjang kawat tersebut adalah:

Δl = (F x l₀) / (A x γ)

= (96 x 10) / (4 x 10⁻⁶ x 25 x 10¹²)

= 9,6 x 10⁶ m

Jika [tex] R = S^T [/tex] , tentukan nilai [tex] x+y [/tex]

Jawab:

[tex] \begin{align} R &= S^T \\ \begin{pmatrix} 5&2\\4&-3\end{pmatrix} &= \begin{pmatrix} 5&\frac{1}{2}x\\ 4y&-3 \end{pmatrix}^T \\ \begin{pmatrix} 5&2\\4&-3\end{pmatrix} &= \begin{pmatrix} 5&4y\\ \frac{1}{2}x&-3 \end{pmatrix} \end{align} [/tex]

Diperoleh dari kesamaan nilai matriks di atas:

[tex] \begin{align} 2&=4y \\ y &= \dfrac{2}{4} = \dfrac{1}{2} \end{align} [/tex]

kemudian:

[tex] \begin{align} 4 &= \dfrac{1}{2}x \\ x &= 4\cdot 2 = 8 \end{align} [/tex]

Sehingga nilai [tex] x+y = 8 + \frac{1}{2} = 8 \frac{1}{2} = \frac{17}{2} [/tex]

8. jelaskan sifat-sifat fisika,kimia,dan biologi dari matriks sitoplasma!


Matriks adalah cairan transparan yang homogen dan bersifat koloid.
Matriks sitoplasma memiliki sifat-sifat fisik sebagai berikut :
a. Efek Tyndall, yakni kemampuan matriks sitoplasma memantulkan cahaya yang pantulannya yang berupa kerucut.
b. Gerak Brown, yakni gerak partikel penyusun larutan koloid yang berupa gerakan zig-zag.
c. Gerak Siklosis, yakni gerak matriks sitoplasma yang berupa gerakan arus.
d. Memiliki tegangan permukaan .
e. Adsorbsi meningkatkan konsentrasi pada tegangan permukaan.
f. Bertindak sebagai larutan buffer atau larutan penyangga. Penyusun matriks sitoplasma meliputi oksigen 62%, karbon 20%, hidrogen 10%, dan nitrogen 3%. Penyusun lain yang jumlahnya rendah adalah Ca 2,5%; P 1,14%; C1 0,16%; S 0,14%; K 0,11%; Na 0,10%; Mg 0,07%; I 0,014%; Fe 0,10%, dan unsur lainnya dalam jumlah yang sangat kecil. Sifat biologis matriks sitoplasma meliputi iritabilitas dan konduktivitas. Iritabilitas, artinya sensitif terhadap rangsangan. Sementara itu, konduktivitas berarti mampu memindahkan rangsangan atau implus.

13. Soal Fisika jawabannya harus fisika​


Jawaban:

C. 38.000 J

Penjelasan:

Diketahui :

massa tembaga (m) = 2 kg

perubahan suhu (ΔT) = 50° C

kalor jenis tembaga (c) = 380 J / kg°C

Ditanyakan : Kalor yang diperlukan (Q)

Jawab :

Rumus kalor : Q = m × c × ΔT

Q = m × c × ΔT

Q = 2 kg × 380 J/kg°C × 50°C

Q = 38.000 J

Jadi, besar kalor yang diperlukan adalah 38.000 J.

Diketahui:

A = [tex]\left[\begin{array}{ccc}1&2&1\\0&3&1\end{array}\right][/tex] , B = [tex]\left[\begin{array}{ccc}1&2&3\\0&1&2\\1&1&2\end{array}\right][/tex] , C = [tex]\left[\begin{array}{ccc}1&2\\2&2\\3&2\end{array}\right][/tex]

a) 2B - 5C

2B = 2  [tex]\left[\begin{array}{ccc}1&2&3\\0&1&2\\1&1&2\end{array}\right][/tex]  =  [tex]\left[\begin{array}{ccc}2&4&6\\0&2&4\\2&2&4\end{array}\right][/tex]5C = 5  [tex]\left[\begin{array}{ccc}1&2\\2&2\\3&2\end{array}\right][/tex]  =  [tex]\left[\begin{array}{ccc}5&10\\10&10\\15&10\end{array}\right][/tex]

2B - 5C tidak dapat diketahui hasilnya karena ordonya berbeda. B memiliki ordo 3x3 sedangkan C memiliki ordo 3x2. Seperti yang diketahui dari sifat penjumlahan dan pengurangan matriks, matriks tidak dapat dijumlahankan atau dikurangkan apabila ordo dari semua pihak tidak sama.

b) AC dan C[tex]B^{t}[/tex]

A = [tex]\left[\begin{array}{ccc}1&2&1\\0&3&1\end{array}\right][/tex] , C = [tex]\left[\begin{array}{ccc}1&2\\2&2\\3&2\end{array}\right][/tex]

A memiliki ordo 2x3 sedangkan C memiliki ordo 3x2, maka sesuai dengan sifat perkalian matriks yaitu (axb) x (bxc) = axc . A dan C memenuhi sifat perkalian matriks.

Menentukan nilai AC

AC = [tex]\left[\begin{array}{ccc}1&2&1\\0&3&1\end{array}\right][/tex] x  [tex]\left[\begin{array}{ccc}1&2\\2&2\\3&2\end{array}\right][/tex]

AC = [tex]\left[\begin{array}{ccc}1x1 + 2x2 + 1x3 &1x2 +2x2 + 1x2\\0x1 + 3x2 + 1x3 &0x2 + 3x2 + 1x2\end{array}\right][/tex]

AC = [tex]\left[\begin{array}{ccc} 8 &8\\10 &8\end{array}\right][/tex]

29. Buatlah program untuk menukarkan dua buah matriks 2x4 menggunakan method dan array dengan ketentuan: a. Terdiri dari 2 method: matriks1 dan matriks2 yang nilai kembaliannnya berupa array. b. Nilai matriks1: 3 1 4 2 5 3 Nilai matriks2: 9 4 6 8 7 1


Jawaban:

Berikut jawabannya dalam bahasa C:

#define _CRT_SECURE_NO_WARNINGS

#include<stdio.h>

void printArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

printf("Matrix 1:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix1[i][j]);

 }

 printf("\n");

}

printf("\nMatrix 2:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix2[i][j]);

 }

 printf("\n");

}

}

void swapArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

int temp;

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  temp = matrix1[i][j];

  matrix1[i][j] = matrix2[i][j];

  matrix2[i][j] = temp;

 }

 

}

}

int main()

{

int cols = 2;

int rows = 3;

int matrix1[3][2] =  

{

 {3,1},

 {4,2},

 {5,3}

};

int matrix2[3][2] =  

{

 {9,4},

 {6,8},

 {7,1}

};

printArray(cols,rows,matrix1,matrix2);

swapArray(cols,rows,matrix1,matrix2);

printf("\nAfter Swap\n\n");

printArray(cols,rows,matrix1,matrix2);

getchar();

return 0;

}

Penjelasan:

Program ini pertama menginisialisasi jumlah baris dan kolom. Kemudian matrix1 dan matrix2 diinisialisasi dengan nilai yang telah ditentukan. Untuk mengetahui perbedaan antara sebelum dan sesudah maka dibuatlah method printArray untuk mencetak matrix1 dan matrix2. Method printArray digunakan 2 kali untuk melihat nilai sebelum dan sesudah. Terakhir ada method untuk swap nilai matrix 1 dan matrix 2 dengan menggunakan bantuan variable temp untuk menyimpan nilai matrix 1 sebelum dioverwrite nilai matrix2 dan digunakan untuk mengoverwrite nilai matrix2.
