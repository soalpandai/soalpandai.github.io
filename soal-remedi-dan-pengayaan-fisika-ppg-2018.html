No. 3 Pengayaan 
-------------------------
Jika :
|F1 + F2| = 2.|F1 - F2| dan F1 = 3.F2

Tanya :
Cos θ = __?

Jawab :

Step 1
---------
|F1 + F2| = √(F1² + F2² + 2.F1.F2.cos θ)

kemudian masukkan F1 = 3.F2, ke persmaan diatas :
|F1 + F2| = √((3.F2)² + F2² + 2.(3.F2).F2.cos θ)
|F1 + F2| = √(9F2² + F2² + 6F2²cos θ)
|F1 + F2| = √(10F2² + 6F2²cos θ) ..... Pers (1)

Step 2
---------
2.|F1 + F2| = 2.√(F1² + F2² - 2.F1.F2.cos θ)

kemudian masukkan F1 = 3.F2, ke persmaan diatas :
2.|F1 + F2| = 2.√(F1² + F2² - 2.F1.F2.cos θ)
2.|F1 + F2| = 2.√((3.F2)² + F2² - 2.(3.F2).F2.cos θ)
2.|F1 + F2| = 2.√(9F2² + F2² - 6F2².cos θ)
2.|F1 + F2| = 2.√(10F2² - 6F2².cos θ) .... Pers (2)


Step 3
----------
Pers (1) sama kan dengan Pers (2)
|F1 + F2| = 2.|F1 - F2|
√(10F2² + 6F2².cos θ) = 2.√(10F2² - 6F2².cos θ)

kemudian semua ruas di kuadratkan menjadi :
(10F2² + 6F2².cos θ) = 4.(10F2² - 6F2².cos θ)
10F2² + 6F2².cos θ = 40F2² - 24F2².cos θ
24F2².cos θ + 6F2²cos θ = 40F2² - 10F2²
30F2²cos θ = 30F2²

cos θ = 1



No. 4 Pengayaan 
-------------------------
Diket : 
Jika F2 = 3.F1
dan sudut θ = 60°

Tanya :
perbandingan Nilai |F1 + F2| / |F1 - F2| = ___?

Jawab :
Step 1
---------
|F1 + F2| = √(F1² + F2² + 2.F1.F2.cos θ)

Masukkan F2 = 3.F1 dan θ = 60°, ke persamaan di  atas menjadi :
|F1 + F2| = √(F1² + (3.F1)² + 2.F1.(3.F1).cos 60°)
|F1 + F2| = √(F1² + 9.F1² + 6.F1².1/2)
|F1 + F2| = √(F1² + 9.F1² + 3.F1²)
|F1 + F2| = √(13.F1²)
|F1 + F2| = F1.√13

Step 2
---------
|F1 - F2| = √(F1² + F2² - 2.F1.F2.cos θ)

Masukkan F2 = 3.F1 dan θ = 60°, ke persamaan di  atas menjadi :
|F1 - F2| = √(F1² + (3.F1)² + 2.F1.(3.F1).cos 60°)
|F1 - F2| = √(F1² + 9.F1² - 6.F1².1/2)
|F1 - F2| = √(F1² + 9.F1² - 3.F1²)
|F1 - F2| = √(7.F1²)
|F1 - F2| = F1.√7

Step 3
---------
Jadi nilai perbandingan dari :
= |F1 + F2| / |F1 - F2|
= F1.√13 / F1.√7
= √13 / √7


N0. 5 Pengayaan 
---------------------------
Diket :
Jika kedua vektor besarnya sama, F1 = F2 = F
Jika Vektor Jumlah dibandingkan Vektor Selesih adalah 1/2,
|F1 + F2| / |F1 - F2| = 1/2

Tanya : 
sudut apit yang dibentuk, θ = __?

Jawab : 

Step 1
---------
|F1 + F2| = √(F1² + F2² + 2.F1.F2.cos θ) 
Masukkan F1 = F2 = F, ke persamaan di  atas menjadi :

|F1 + F2| = √(F1² + F2² + 2.F1.F2.cos θ) 
|F1 + F2| = √(F² + F² + 2.F.F.cos θ)
|F1 + F2| = √(F² + F² + 2F².cos θ)
|F1 + F2| = √(2F² + 2F².cos θ)  ........ Pers (1)

Step 2
---------
|F1 - F2| = √(F1² + F2² - 2.F1.F2.cos θ) 
Masukkan F1 = F2 = F, ke persamaan di  atas menjadi :

|F1 - F2| = √(F1² + F2² - 2.F1.F2.cos θ) 
|F1 - F2| = √(F² + F² - 2.F.F.cos θ)
|F1 - F2| = √(F² + F² - 2F².cos θ)
|F1 - F2| = √(2F² - 2F².cos θ)  ........ Pers (2)


Step 3
---------
Pers (1) bandingkan dengan Pers (2) , menjadi :

|F1 + F2| = √(2F² + 2F².cos θ)
-----------------------------------------
|F1 - F2| = √(2F² - 2F².cos θ)

1/2 = √(2F² + 2F².cos θ) / √(2F² - 2F².cos θ)

semua ruas dikuadratkan menjadi :

1/4 = 2F² + 2F².cos θ / 2F² - 2F².cos θ

4.(2F² + 2F².cos θ) = 1. (2F² - 2F².cos θ)

8F² + 8F².cos θ = 2F² - 2F².cos θ

2F².cos θ + 8F².cos θ = 2F² - 8F²

10F².cos θ = - 6F²

cos θ = - 0,6

θ = 127° atau θ = 233°
