2. Bayangan dari x = -2 adalah -9

3. Nilai c adalah -5

4. Sumbu simetrinya adalah -1

Penjelasan dengan langkah-langkah:

» No. 1

Bentuk umum persamaan kuadrat ax² + bx + c = 0, dimana:

a = koefisien dari x²b = koefisien dari xc = konstanta

Maka dari persamaan -7n² - n + 2 = 0, didapat koefisien n² adalah -7

┄

» No. 2

f(x) = x² + 3x - 7

Bayangan dari x = -2:

f(-2) = (-2)² + 3(-2) - 7

        = 4 - 6 - 7

        = -2 - 7

        = -9

┄

» No. 3

2x² - 9x + c = 0

a = 2, b = -9, c = c

D = b² - 4ac

121 = (-9)² - 4(2)(c)

121 = 81 - 8c

8c = 81 - 121

8c = -40

  c = -40/8

  c = -5

┄

» No. 4

f(x) = 2x² + 4x - 6

a = 2, b = 4, c = -6

Sumbu simetrinya:

x = -b/2a

   = -4/2(2)

   = -4/4

   = -1

10. MATEMATIKA(perkalian dua vektor)___________#jangan ngasal#harus pake cara# soal di foto# matematika asik​


a. Nilai dari [tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q} }[/tex] adalah 4.

b. Nilai dari [tex]\vec{p}.(-2\vec{q})[/tex] adalah -24.

PEMBAHASAN

Vektor adalah suatu besaran yang memiliki nilai dan arah. Salah satu operasi pada vektor adalah dot product, dimana hasil dot product dari vektor [tex]\vec{a}=(a_1,a_2,a_3,...,a_n)[/tex] dan vektor [tex]\vec{b}=(b_1,b_2,b_3,...,b_n)[/tex] adalah :

[tex]\vec{a}.\vec{b}=a_1b_2+a_2b_2+a_3b_3+...+a_nb_n[/tex]

.

DIKETAHUI

[tex]\vec{p}=\begin{pmatrix}6\\ -3\end{pmatrix}~~\vec{q}=\begin{pmatrix}1\\ -2\end{pmatrix}[/tex]

.

DITANYA

Tentukan nilai dari :

a. [tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q}}[/tex]

b. [tex]\vec{p}.(-2\vec{q})[/tex]

.

PENYELESAIAN

Soal a.

[tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q}=\frac{1}{3}\begin{pmatrix}6\\ -3\end{pmatrix}.\begin{pmatrix}1\\ -2\end{pmatrix} }[/tex]

[tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q}=\begin{pmatrix}\frac{6}{3}\\ -\frac{3}{3}\end{pmatrix}.\begin{pmatrix}1\\ -2\end{pmatrix} }[/tex]

[tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q}=\begin{pmatrix}2\\ -1\end{pmatrix}.\begin{pmatrix}1\\ -2\end{pmatrix} }[/tex]

[tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q}=2(1)-1(-2) }[/tex]

[tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q}=2+2 }[/tex]

[tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q}=4 }[/tex]

.

Soal b.

[tex]\vec{p}.(-2\vec{q})=\begin{pmatrix}6\\ -3\end{pmatrix}.\left [ -2\begin{pmatrix}1\\ -2\end{pmatrix} \right ][/tex]

[tex]\vec{p}.(-2\vec{q})=\begin{pmatrix}6\\ -3\end{pmatrix}.\begin{pmatrix}-2\times1\\ -2\times-2\end{pmatrix}[/tex]

[tex]\vec{p}.(-2\vec{q})=\begin{pmatrix}6\\ -3\end{pmatrix}.\begin{pmatrix}-2\\ 4\end{pmatrix}[/tex]

[tex]\vec{p}.(-2\vec{q})=6(-2)-3(4)[/tex]

[tex]\vec{p}.(-2\vec{q})=-12-12[/tex]

[tex]\vec{p}.(-2\vec{q})=-24[/tex]

.

KESIMPULAN

a. Nilai dari [tex]\displaystyle{\frac{1}{3}\vec{p}.\vec{q} }[/tex] adalah 4.

b. Nilai dari [tex]\vec{p}.(-2\vec{q})[/tex] adalah -24.

.

-1, -10,0,8,9 || bilangan bulat adalah bilangan yang terdiri atas bilangan negatif,nol, dan positif (cacah).

Pembahasan :

bilangan bulat adalah bilangan yang terdiri atas bilangan negatif,nol, dan positif (cacah). seperti pengertiannya bilangan cacah dibagi menjadi 3 yaitu :

Bilangan negatif

Bilangan negatif memiliki lambang (-) didepan angka. lihatlah pada gambar! Bilangan negatif memiliki peraturan yaitu : semakin kekiri makan semakin kecil, sebaliknya semakin kekanan makan semakin besar. sebagai contoh ada 2 bilangan yaitu -10 dan -2 ,maka yang lebih besar adalah -2. lambang (-) dibaca negatif, contohnya :

a. -70 dibaca negatif tujuh puluh

b. -9 dibaca negatif sembilan

bilangan negatif terdiri atas -1 , -2 , -3 .....

Bilangan nol

bilangan nol tidak masuk pada negatif ataupun positif. Bilangan nol hanya satu yaitu 0

Bilangan positif

Bilangan positif memiliki lambang (+) ,lambang ini boleh ditulis atau tidak. lihatlah pada gambar! bilangan positif terdiri atas 1,2,3,4,5,6,7,..... . bilangan positif ini memiliki peraturan yg sama dengan bilangan negatif. bilangan bulat dibaca :

a. 90 Dibaca positif sembilan puluh / sembilan puluh

b. 5 Dibaca positif lima / lima
