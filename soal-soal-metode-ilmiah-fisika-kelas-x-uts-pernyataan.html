15. Soal UTS fisika kelas 11 K13, Mohon tidak asal asalan. Nomor 7 dan 8


KESEIMBANGAN BENDA TEGAR (kategori 6)

nomor 7
r1 = R
I1 = I
r2 = 2R
I2?

penyelesaian:
I2 = I1 r2² / r1²
I2 = I . (2R²) / R²
I2 = I . 4R² / R² (coret R²)
I2 = I . 4
I2 = 4I

nomor 8
m = 1 kg
R = 20 cm = 0,2 m
k = 2/3 (konstanta inersia)
I?

penyelesaian:
I = k m R²
I = 2/3 . 1 . 0,2²
I = 2/3 . 0,04
I = 0,027 kgm²7)
R1 = a
I1 = b
R2 = 2R1 = 2a
I2 = .....

rumus :
R1² : R2² = I1 : I2

jawab :
a² : (2a)² = b : I2
a² x i2 = 4a² x b
i2 = 4a²b/a²
i2 = 4b

8)
m = 1 kg
r = 20 cm = 20/100 = 0,2 m
konstanta (k) = 0,667 = ⅔
I = .....

rumus :
I = k x m x r²

jawab :
I = ⅔ x 1 x (0,2)² = ⅔ x 0,04
I = ⅔ x 4/100
I = 8/300
I = 0,0267 kgm²
