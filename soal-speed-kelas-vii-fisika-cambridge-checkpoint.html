11. Mapel: FisikaKelas: VIII Materi: Tekanan Zat Cair - Tekanan Hidrostatis​


Sebelumnya saya akan memberikan satu rumus yang dapat menyelesaikan permasalahan ini. Rumus itu dinamakan tekanan hidrostatis:

[tex]\boxed{\sf P = \rho ~.~g~.~h}[/tex]

Keterangan:

[tex]\rho[/tex] = massa jenis zat.

g = percepatan gravitasi.

h = kedalaman.

[tex]~[/tex]

Kita mulai dari zat X:

[tex]\sf P = \rho ~.~g~.~h[/tex]

[tex]\sf 450~N/m^2 = \rho ~.~10~m/s^2~.~5~cm[/tex]

[tex]\sf 450~N/m^2 = \rho ~.~10~m/s^2~.~0,05~m[/tex]

[tex]\sf 450~N/m^2 = \rho ~.~0,5~m^2/s^2[/tex]

[tex]\sf \rho = \frac{450~N/m^2}{0,5~m^2/s^2}[/tex]

[tex]\sf \rho = 900 kg/m^3[/tex]

[tex]\sf \rho = 0,9~g/cm^3[/tex]

Maka zat X = minyak zaitun.

[tex]~[/tex]

Lanjutkan ke zat Y:

[tex]\sf P = \rho ~.~g~.~h[/tex]

[tex]\sf 441~N/m^2 = \rho ~.~10~m/s^2~.~3,5~cm[/tex]

[tex]\sf 441~N/m^2 = \rho ~.~10~m/s^2~.~0,035~m[/tex]

[tex]\sf 441~N/m^2 = \rho ~.~0,35~m^2/s^2[/tex]

[tex]\sf \rho = \frac{441~N/m^2}{0,35~m^2/s^2}[/tex]

[tex]\sf \rho = 1.260 kg/m^3[/tex]

[tex]\sf \rho = 1,26~g/cm^3[/tex]

Maka zat Y = gliserin.

[tex]~[/tex]

Terakhir zat Z:

[tex]\sf P = \rho ~.~g~.~h[/tex]

[tex]\sf 440~N/m^2 = \rho ~.~10~m/s^2~.~5,5~cm[/tex]

[tex]\sf 440~N/m^2 = \rho ~.~10~m/s^2~.~0,055~m[/tex]

[tex]\sf 440~N/m^2 = \rho ~.~0,55~m^2/s^2[/tex]

[tex]\sf \rho = \frac{440~N/m^2}{0,55~m^2/s^2}[/tex]

[tex]\sf \rho = 800 kg/m^3[/tex]

[tex]\sf \rho = 0,8~g/cm^3[/tex]

Maka zat Z = bensin.

[tex]~[/tex]

Kesimpulan

Opsi yang benar C.
