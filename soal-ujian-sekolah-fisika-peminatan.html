9. alasan memilih peminatan matematika dan ilmu pengetahuan alam, terdiri dari mata pelajaran matematika, fisika, kimia, dan biologi


Alasan memilih peminatan matematika dan ilmu pengetahuan alam, terdiri dari mata pelajaran matematika, fisika, kimia, dan biologi. Simak lebih detil dalam pembahasan.

Pembahasan

Banyak alasan yang bisa dikemukakan oleh setiap siswa dalam memilih peminatan. Tentu saja, setiap siswa pun pasti memiliki alasan yang berbeda dalam memilih jurusan baik IPA ataupun IPS. Alasan tersebut bisa dilihat dari:

1. Minat

Kita harus melihat apakah kita minat atau memiliki passion terhadap hitungan atau hafalan. Jika kita memiliki minat atau passion terhadap hitungan, berarti kita bisa memilih IPA, namun sebaliknya kita bisa memilih IPS.

2. Kemampuan

Selain minat, kita juga harus mengukur kemampuan kita. Kalau dirasa pas-pasan sebaiknya jangan memilih IPA. Tapi kalau yakin dan merasa bisa meningkatkan kemampuan karena kita sudah minat, silakan saja memilih IPA.

3. Jurusan kuliah

Selain melihat minat dan mengukur kemampuan, kita juga bisa melihat saat kuliah, kita mau masuk jurusan apa. Kalau kita mau masuk jurusan kedokteran, teknik, dan MIPA, itu sudah pasti kita harus mengambil peminatan jurusan IPA. Begitu pula jurusan lainnya juga menyesuaikan.

4. Teman

Terkadang, ada juga beberapa siswa memilih peminatan karena mengikuti teman, tapi ini tidaklah disarankan. Sebaiknya kita mempunyai alasan tersendiri jangan hanya karena teman kita memilih peminatan tersebut tapi begitu menjalani, kita sendiri kerepotan mengikuti peminatan yang dipilih.

5. Orangtua

Tidak sedikit pula orang memilih peminatan tertentu karena tuntutan orang tua. Kita boleh saja mengikuti tuntutan orang tua selama kita mampu. Namun, kalau tidak, beri pengertian yang baik terhadap orang tua. Orang tua pasti akan memahami kita.

23. Tolong bagaimana mengerjakan soal-soal peminatan ini yang dilingkari bantuin ya ??


Materi : Trigonometri
Bahasan : Persamaan Trigonometri

Langkah - langkah :
(6). Sin x + 1/2(√3) = 0, untuk 0 ≤ x ≤ 2π
sin x = - 1/2(√3)
sin x = 4π/3

- Mencari x dengan α + k(360)
Untuk k = 0
Sin x = 4π/3 + 0
Sin x = 4π/3
Untuk k = 1
Sin x = 4π/3 + 2π
Sin x = 10π/3 (T.M)

- Mencari x dengan (180 - α) + k(360)
Untuk k = 0
x = π - 4π/3
x = - π/3 (T. M)
Untuk k = 1
x = (π - 4π/3) + 2π
x = 5π/3
HP = {4π/3, 5π/3}

(7). Cos x = (1/2)√2 untuk 0 ≤ x ≤ 360°
x = 45°
Mencari x dengan α + k(360)
- Untuk k = 0
x = 45°
- Untuk k = 1
x = 45° + 360
x = 405° (T.M)

Mencari x dengan - a + k(360)
Untuk k = 0
x = - 45° (T.M)
Untuk x = 1
x = - 45° + 360
x = 315°
HP = {45°, 315°}
