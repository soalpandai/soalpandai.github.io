9. geteran bunyi merambat melalui zat padat, cair dan​


Bunyi atau suara adalah gelombang longitudinal yang merambat melalui medium. Medium ini dapat berupa zat cair, padat, gas.

Pembahasan

Berdasarkan frekuensinya bunyi dibedakan menjadi tiga jenis, yaitu:

1. Bunyi Infrasonik adalah bunyi yang mempunyai frekuensi sangat rendah, yaitu kurang dari 20 Hz. Bunyi infrasonik ini dapat didengar oleh kelelawar, jang***k dan kuda.

2. Bunyi Audiosonik adalah bunyi yang mempunyai frekuensi di antara 20-20.000 Hz. Bunyi audiosonik ini dapat didengar oleh manusia. Bunyi kereta api yang melintas akan berbeda dengan bunyi burung yang berkicau.

3. Bunyi Ultrasonik adalah bunyi yang mempunyai frekuensi sangat tinggi, yaitu lebih dari 20.000 Hz. Bunyi ultrasonik ini dapat didengar oleh lumba-lumba.

Bunyi merambat pada 3 medium atau zat perantara, sebagai berikut:

a. Melalui benda padat

Bunyi dapat merambat dengan getaran bunyi yang merambat pada benda padat seperti regangan tali antara dua benda misalnya membuat mainan telepon.

b. Melalui benda cair

Getaran bunyi akan mengalir melalui zat cair dan memantul kembali ketempat asal bunyi tersebut. Misalnya bunyi yang dapat merambat melalui zat cair oleh tim SAR untuk mencari dan menolong kecelakaan yang terjadi di tengah lautan.

c. Melalui benda gas

Getaran bunyi merambat memlaui udara Misalnya guntur dapat kita dengar karena getaran suaranya masuk ke telinga kita setelah merambat melalui udara.

39. sebuah ayunan bergetar 80kali dalam 20s,tentukan periode geteran dan frekuensi ayunan!


T = t/n = 20/80 = 0,25 sekon

f = n/t = 80/20 = 4 Hz

Sebuah ayunan bergetar 80 kali dalam 20 s, maka periode getaran dan frekuensi ayunan adalah 1/4 sekon dan 4 Hz

PEMBAHASAN

Gelombang sinusoidal adalah gelombang yang membentuk fungsi sinus. Beberapa rumus dasar gelombang yang perlu di ingat adalah sebagai berikut ini :

f = 1 / T

v = λ . f

Δx / λ = ΔΘ / 2π

Δt / T = ΔΘ / 2π

dimana :

f = frekuensi gelombang ( Hz )

T = periode gelombang ( s )

v = cepat rambat gelombang ( m/s )

λ = panjang gelombang ( m )

Δx = jarak pisah antar dua titik pada gelombang ( m )

Δt = selang waktu ( s )

ΔΘ = beda fase ( radian )

Bentuk persaman gelombang adalah :

y = A sin ( ωt - kx )

dimana :

y = simpangan gelombang ( m )

A = amplitudo gelombang ( m )

ω = frekuensi sudut ( rad/s )

t = waktu ( s )

k = bilangan gelombang ( 1/m ) = 2π/λ

x = jarak titik dari sumber ( m )

Okay mari kita gunakan rumus ini untuk menyelesaikan soalnya.

Diketahui :

n = 80

t = 20 s

Ditanyakan :

T = ?

f = ?

Penyelesaian :

T = t / n

T = 20/80

T = 1/4 sekon

f = 1/T

f = 1/ (1/4)

f = 4 Hz
