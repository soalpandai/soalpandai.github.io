Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5

27. tolong bantuin soal un tahun 2013


Ingat rumus:
Lim x → ∞  (√(ax² + bx + c) - √(px²  + qx + r))
= (b - q)/(2√a) ----->> (Jika a = p)
= ∞ ------------------->> (Jika a > p)
= -∞ ------------------>> (Jika a < p)

Penyelesaian:
Lim x → ∞     (3x - 1 - √(9x²  + 3x - 5))
= Lim x → ∞  (√(3x - 1)² - √(9x²  + 3x - 5))
= Lim x → ∞  (√(9x² - 6x + 1) - √(9x²  + 3x - 5))
= (b - q)/(2√a)
= (-6 - 3)/(2√9)
= -9/(2.3)
= -9/6

A. Mencari nilai [OH-]
--> Soal UN ini mengaitkan 2 bab (kyk SBMPTN, ya), yaitu hidrolisis garam dan hasil kali kelarutan (Ksp), sehingga:
     CH3COOH + NaOH --> CH3COONa + H2O
m        0.8           0,8
b        -0.8          -0.8              +0.8          +0.8
s           -              -                 0.8

[G] = 0.8 mmol / 200 mL = 4 x 10^-3

Bersisa garamnya sebesar 0,8 mmol, masukkan ke rumus hidrolisis:
[OH-] = √Kw x [G]/Ka
         = √10^-14 x 4 x 10^-3 / 10^-5
         = 2 x 10^-6

B. Mencari [Fe2+] dari Fe(OH)2
--> Untuk mencarinya, buat reaksi ionisasi Besi (II) hidroksida tsb:
Fe(OH)2 ⇄ Fe2+ + 2OH-

1,6 x 10^-15 = [Fe2+] [2 x 10^-6]^2
