6. tolong jawab,mapel ipa fisika...kls 9makasih..​


diket:

q₁ = 180 μC = 180 x 10⁻⁶ C

q₂ = 320 μC = 320 x 10⁻⁶ C

jarak (r) = 40 cm = 4 x 10⁻¹ m

ditanya:

gaya coulomb (F)....?

jawab:

F = (k q₁ q₂) / r²

F = (9 x 10⁹ Nm²/C² x 180 x 10⁻⁶ C x 320 x 10⁻⁶C) / (4 x 10⁻¹ m)²

  = (518.400 x 10⁹⁻⁶⁻⁶) / (16 x 10⁻²)

  = (518.400 x 10⁻³) / (16 x 10⁻²)

  = 32.400 x 10⁻³⁻⁽⁻²⁾ N

  = 32.400 x 10⁻¹ N

  = 3.240 N

Jadi,

- besar gaya coulomb adalah 3.240 N.

- jenis interaksi muatan adalah tolak menolak (karena keduanya positif).

11. soal fisika latihan usbn mohon dibantu


RELATIVITAS

Soal nomor (1)

Diketahui:

Kecepatan pesawat A menurut pengamat di bumi, vap = 0,5c

Kecepatan pesawat B menurut pengamat di bumi, vbp = -0,4c (arah berlawanan dengan pesawat A)

Ditanya:

a. Kecepatan pesawat A menurut pilot pesawat B, vab = ...?

b. Kecepatan pesawat B menurut pilot pesawat A, vba = ...?

Pembahasan:

a. Kecepatan pesawat A menurut pilot pesawat B

vab = (vap + vpb)/(1+vap.vpb/c²)

vab = (0,5c + 0,4c)/(1+0,5c.0,4c/c²)

vab = 0,9c/1,2

vab = 3c/4

b. Kecepatan pesawat B menurut pilot pesawat A

vba = (vbp + vpa)/(1+vbp.vpa/c²)

vab = [-0,4c + (-0,5c)]/[1+(-0,4c).(-0,5c)/c²]

vab = -0,9c/1,2

vab = -3c/4

Soal nomor (2)

Diketahui:

Usia awal A, ta1 = 10 tahun

Usia awal B, tb1 = 10 tahun

Usia akhir A, ta2 = 40 tahun

Ditanya:

Usia B menurut A, tb' = ...?

Pembahasan

Selisih usia B menurut pengamat di bumi

Δtb = 40 - 10 = 30 tahun

Selisih usia B menurut A

Δtb' = Δtb/√(1 - v²/c²)

Δtb' = 30/√[1 - (0,8c)²/c²]

Δtb' = 30/√(1 - 0,64)

Δtb' = 30/√0,36

Δtb' = 30/0,6

Δtb' = 50

tb' = 10 + 50 = 60 tahun

Jadi usia B menurut A adalah 60 tahun

Soal nomor (3)

Diketahui:

Kecepatan pesawat 1 menurut pengamat di bumi, v13 = 0,5c

Kecepatan pesawat 2 menurut pengamat di bumi, v23 = -0,25c (arah berlawanan dengan pesawat 1)

Ditanya:

a. Kecepatan pesawat 1 menurut pilot pesawat 2, v12 = ...?

b. Kecepatan pesawat 2 menurut pilot pesawat 1, v21 = ...?

Pembahasan:

a. Kecepatan pesawat 1 menurut pilot pesawat 2

v12 = (v13 + v32)/(1+v13.v32/c²)

v12 = (0,5c + 0,25c)/(1+0,5c.0,25c/c²)

v12 = 0,75c/1,125

v12 = c/15

b. Kecepatan pesawat 2 menurut pilot pesawat 1

v21 = (v23 + v31)/(1+v23.v31/c²)

v21 = [-0,25c + (-0,5c)]/[1+(-0,25c).(-0,5c)/c²]

v21 = -0,75c/1,125

v21 = -c/15

Soal nomor (4)

Diketahui:

Panjang batang diamati saat diam, Lo = 1 m

Panjang batang diamati saat bergerak, L = 0,8 m

Ditanya:

Hitung kecepatan gerak batang, v = ...?

Pembahasan:

L = Lo.√(1 - v²/c²)

0,8 = 1.√(1 - v²/c²)

0,8 = √(1 - v²/c²)

0,64 = 1 - v²/c²

v²/c² = 1 - 0,64

v²/c² = 0,36

v² = 0,36c²

v = 0,6c

Untuk menghitung suhu di puncak gunung, kita dapat menggunakan rumus penurunan suhu dengan ketinggian:

ΔT = 0,6 °C per 100 m

Dalam hal ini, perbedaan ketinggian antara permukaan laut dan puncak gunung adalah 2800 m. Sehingga:

ΔT = 0,6 °C x 28 = 16,8 °C

Suhu di puncak gunung adalah suhu di permukaan laut dikurangi penurunan suhu tersebut:

Suhu di puncak gunung = 27 °C - 16,8 °C = 10,2 °C

Untuk menghitung suhu di lereng gunung, kita perlu mengetahui suhu di daerah bayangan hujan (adiabatik basah). Kita dapat menggunakan rumus adiabatik basah untuk menghitung perubahan suhu dalam udara yang naik atau turun:

ΔT = (L / Cpd) x (e2 - e1)

Di mana:

ΔT = perubahan suhu (dalam derajat Celsius)

L = kalor laten penguapan air (2,5 x 10^6 J/kg)

Cpd = kapasitas kalor udara kering pada tekanan tetap (1005 J/kg·K)

e2 = tekanan uap jenuh pada suhu di daerah bayangan hujan

e1 = tekanan uap aktual pada suhu di puncak gunung

Misalkan suhu di daerah bayangan hujan adalah 15 °C, maka tekanan uap aktual pada suhu di puncak gunung dapat dihitung menggunakan rumus Clausius-Clapeyron:

e1 = e2 x exp[-L / (Rv x Cp) x (1 / T2 - 1 / T1)]

Di mana:

Rv = konstanta gas uap (461,5 J/kg·K)

Cp = kapasitas kalor pada tekanan tetap untuk udara kering (1005 J/kg·K)

T2 = suhu di puncak gunung (10,2 °C + 273,15 = 283,35 K)

T1 = suhu di daerah bayangan hujan (15 °C + 273,15 = 288,15 K)

e2 dapat dihitung dengan menggunakan tabel tekanan uap jenuh air pada suhu tertentu. Misalkan e2 pada suhu 15 °C adalah 17,5 mb (millibar):

e1 = 17,5 mb x exp[-2,5 x 10^6 / (461,5 x 1005) x (1 / 283,35 - 1 / 288,15)] = 4,2 mb
